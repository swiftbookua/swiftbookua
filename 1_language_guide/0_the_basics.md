### –ú–µ–∂—ñ —Ü—ñ–ª–∏—Ö —á–∏—Å–µ–ª

–î–æ—Å—Ç—É–ø–∏—Ç–∏—Å—å –¥–æ –Ω–∞–π–±—ñ–ª—å—à–æ–≥–æ —ñ –Ω–∞–π–º–µ–Ω—à–æ–≥–æ –º–æ–∂–ª–∏–≤–æ–≥–æ –∑–Ω–∞—á–µ–Ω–Ω—è –∫–æ–∂–Ω–æ–≥–æ –∑ —Ü—ñ–ª–∏—Ö —Ç–∏–ø—ñ–≤ –º–æ–∂–Ω–∞ –∑–∞ –≤–ª–∞—Å—Ç–∏–≤–æ—Å—Ç—è–º–∏ `min` —Ç–∞ `max`:

–ó–Ω–∞—á–µ–Ω–Ω—è —Ü–∏—Ö –≤–ª–∞—Å—Ç–∏–≤–æ—Å—Ç–µ–π –º–∞—é—Ç—å —á–∏—Å–ª–æ–≤–∏–π —Ç–∏–ø –≤—ñ–¥–ø–æ–≤—ñ–¥–Ω–æ–≥–æ —Ä–æ–∑–º—ñ—Ä—É \(—Ç–∞–∫–∏–π —è–∫ `UInt8` —É –ø—Ä–∏–∫–ª–∞–¥—ñ –≤–∏—â–µ\), —ñ –º–æ–∂—É—Ç—å –¥–∞–ª—ñ –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É–≤–∞—Ç–∏—Å—å —É –≤–∏—Ä–∞–∑–∞—Ö —Ä–∞–∑–æ–º –∑ —ñ–Ω—à–∏–º–∏ –∑–Ω–∞—á–µ–Ω–Ω—è–º–∏ —Ç–æ–≥–æ –∂ —Ç–∏–ø—É.

### Int

–£ –±—ñ–ª—å—à–æ—Å—Ç—ñ –≤–∏–ø–∞–¥–∫—ñ–≤ –Ω–µ –ø–æ—Ç—Ä—ñ–±–Ω–æ –æ–±–∏—Ä–∞—Ç–∏ –∫–æ–Ω–∫—Ä–µ—Ç–Ω–∏–π —Ä–æ–∑–º—ñ—Ä —Ü—ñ–ª–æ—á–∏—Å–µ–ª—å–Ω–æ–≥–æ —Ç–∏–ø—É. –ú–æ–≤–∞ Swift –Ω–∞–¥–∞—î –¥–æ–¥–∞—Ç–∫–æ–≤–∏–π —Ü—ñ–ª–æ—á–∏—Å–µ–ª—å–Ω–∏–π —Ç–∏–ø `Int`, —â–æ –º–∞—î —Ç–∞–∫–∏–π –∂–µ —Ä–æ–∑–º—ñ—Ä, —â–æ —ñ —Ä–æ–∑–º—ñ—Ä –º–∞—à–∏–Ω–Ω–æ–≥–æ —Å–ª–æ–≤–∞ –Ω–∞ –ø–æ—Ç–æ—á–Ω—ñ–π –ø–ª–∞—Ç—Ñ–æ—Ä–º—ñ.

### UInt

## –ß–∏—Å–ª–∞ –∑ —Ä—É—Ö–æ–º–æ—é –∫–æ–º–æ—é

–ß–∏—Å–ª–∞ –∑ —Ä—É—Ö–æ–º–æ—é –∫–æ–º–æ—é –º–æ–∂—É—Ç—å –ø—Ä–µ–¥—Å—Ç–∞–≤–∏—Ç–∏ –Ω–∞–±–∞–≥–∞—Ç–æ –±—ñ–ª—å—à–∏–π –¥—ñ–∞–ø–∞–∑–æ–Ω –∑–Ω–∞—á–µ–Ω—å, –Ω—ñ–∂ —Ü—ñ–ª–æ—á–∏—Å–µ–ª—å–Ω—ñ —Ç–∏–ø–∏, —ñ –º–æ–∂–µ –∑–±–µ—Ä—ñ–≥–∞—Ç–∏ —á–∏—Å–ª–∞, —â–æ –Ω–∞–±–∞–≥–∞—Ç–æ –±—ñ–ª—å—à—ñ —á–∏ –º–µ–Ω—à—ñ –Ω—ñ–∂ —Ç—ñ, —â–æ –º–æ–∂—É—Ç—å –∑–±–µ—Ä—ñ–≥–∞—Ç–∏—Å—å —É —Ç–∏–ø—ñ `Int`. –ú–æ–≤–∞ Swift –Ω–∞–¥–∞—î –¥–≤–∞ –∑–Ω–∞–∫–æ–≤–∏—Ö —Ç–∏–ø–∏ –¥–ª—è —á–∏—Å–µ–ª –∑ —Ä—É—Ö–æ–º–æ—é –∫–æ–º–æ—é:

> **–ü—Ä–∏–º—ñ—Ç–∫–∞**

–°–ª—ñ–¥ –≤–∂–∏–≤–∞—Ç–∏ —Ç–∏–ø `Int` –¥–ª—è –≤—Å—ñ—Ö —Ü—ñ–ª–æ—á–∏—Å–µ–ª—å–Ω–∏—Ö –∫–æ–Ω—Å—Ç–∞–Ω—Ç —ñ –∑–º—ñ–Ω–Ω–∏—Ö –∑–∞–≥–∞–ª—å–Ω–æ–≥–æ –ø—Ä–∏–∑–Ω–∞—á–µ–Ω–Ω—è, –Ω–∞–≤—ñ—Ç—å —è–∫—â–æ –≤—ñ–¥–æ–º–æ, —â–æ –≤–æ–Ω–∏ —Ç–æ—á–Ω–æ –¥–æ–¥–∞—Ç–Ω—ñ. –í–∂–∏–≤–∞—é—á–∏ —Ü—ñ–ª–æ—á–∏—Å–µ–ª—å–Ω–∏–π —Ç–∏–ø –∑–∞ –∑–∞–º–æ–≤—á—É–≤–∞–Ω–Ω—è–º –≤ —â–æ–¥–µ–Ω–Ω–∏—Ö —Å–∏—Ç—É–∞—Ü—ñ—è—Ö –º–∏ –æ—Ç—Ä–∏–º—É—î–º–æ –∫–æ–Ω—Å—Ç–∞–Ω—Ç–∏ —Ç–∞ –∑–º—ñ–Ω–Ω—ñ, —è–∫—ñ —î –æ–¥—Ä–∞–∑—É —Å—É–º—ñ—Å–Ω—ñ —É –∫–æ–¥—ñ —Ç–∞ –ø—ñ–¥—Ö–æ–¥—è—Ç—å –¥–æ —Ç–∏–ø—É, —â–æ –≤–∏–≤–æ–¥–∏—Ç—å—Å—è –∑ —Ü—ñ–ª–æ—á–∏—Å–µ–ª—å–Ω–∏—Ö –ª—ñ—Ç–µ—Ä–∞–ª—å–Ω–∏—Ö –∑–Ω–∞—á–µ–Ω—å.

–°–ª—ñ–¥ –∫–æ—Ä–∏—Å—Ç—É–≤–∞—Ç–∏—Å—å —ñ–Ω—à–∏–º–∏ —Ü—ñ–ª–æ—á–∏—Å–µ–ª—å–Ω–∏–º–∏ —Ç–∏–ø–∞–º–∏ —Ç—ñ–ª—å–∫–∏ —Ç–æ–¥—ñ, –∫–æ–ª–∏ –¥–ª—è –∑–∞–¥–∞—á—ñ –ø–æ—Ç—Ä—ñ–±–Ω—ñ –∫–æ–Ω–∫—Ä–µ—Ç–Ω–æ —Å–∞–º–∏ –≤–æ–Ω–∏. –ù–∞–ø—Ä–∏–∫–ª–∞–¥, –¥–ª—è –æ–±—Ä–æ–±–∫–∏ –¥–∞–Ω–∏—Ö —è–≤–Ω–æ–≥–æ —Ä–æ–∑–º—ñ—Ä—É —ñ–∑ –∑–æ–≤–Ω—ñ—à–Ω—å–æ–≥–æ –¥–∂–µ—Ä–µ–ª–∞, –∞–±–æ –¥–ª—è –æ–ø—Ç–∏–º—ñ–∑–∞—Ü—ñ—ó —à–≤–∏–¥–∫–æ–¥—ñ—ó, –ø–∞–º'—è—Ç—ñ —á–∏ —ñ–Ω—à–æ—ó –Ω–µ–æ–±—Ö—ñ–¥–Ω–æ—ó –æ–ø—Ç–∏–º—ñ–∑–∞—Ü—ñ—ó. –ö–æ—Ä–∏—Å—Ç—É–≤–∞–Ω–Ω—è —Ç–∏–ø–∞–º–∏ —è–≤–Ω–æ–≥–æ —Ä–æ–∑–º—ñ—Ä—É —É —Ü–∏—Ö —Å–∏—Ç—É–∞—Ü—ñ—è—Ö –¥–æ–ø–æ–º–∞–≥–∞—î –≤—ñ–¥–ª–æ–≤–∏—Ç–∏ –±—É–¥—å-—è–∫—ñ –Ω–µ—Å–ø–æ–¥—ñ–≤–∞–Ω—ñ –ø–µ—Ä–µ–ø–æ–≤–Ω–µ–Ω–Ω—è –∑–Ω–∞—á–µ–Ω—å —ñ –Ω–µ—è–≤–Ω–æ –¥–æ–∫—É–º–µ–Ω—Ç—É—î –ø—Ä–∏—Ä–æ–¥—É —ñ–Ω—Ñ–æ—Ä–º–∞—Ü—ñ—ó, —è–∫–∞ –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É—î—Ç—å—Å—è.

### –ü–µ—Ä–µ—Ç–≤–æ—Ä–µ–Ω–Ω—è —Ü—ñ–ª–∏—Ö —á–∏—Å–µ–ª

–î—ñ–∞–ø–∞–∑–æ–Ω —á–∏—Å–µ–ª, —â–æ –º–æ–∂—É—Ç—å –∑–±–µ—Ä—ñ–≥–∞—Ç–∏—Å—å —É —Ü—ñ–ª–æ—á–∏—Å–µ–ª—å–Ω—ñ–π –∫–æ–Ω—Å—Ç–∞–Ω—Ç—ñ —á–∏ –∑–º—ñ–Ω–Ω—ñ–π —î —Ä—ñ–∑–Ω–∏–º –¥–ª—è –∫–æ–∂–Ω–æ–≥–æ —á–∏—Å–µ–ª—å–Ω–æ–≥–æ —Ç–∏–ø—É. –ö–æ–Ω—Å—Ç–∞–Ω—Ç–∞ —á–∏ –∑–º—ñ–Ω–Ω–∞ —Ç–∏–ø—É `Int8` –º–æ–∂–µ –∑–±–µ—Ä—ñ–≥–∞—Ç–∏ –∑–Ω–∞—á–µ–Ω–Ω—è–º –º—ñ–∂ `-128` —Ç–∞ `127`, —Ç–æ–¥—ñ —è–∫ –∫–æ–Ω—Å—Ç–∞–Ω—Ç–∞ —á–∏ –∑–º—ñ–Ω–Ω–∞ —Ç–∏–ø—É `UInt8` –º–æ–∂–µ –∑–±–µ—Ä—ñ–≥–∞—Ç–∏ –∑–Ω–∞—á–µ–Ω–Ω—è–º –º—ñ–∂ `0` —Ç–∞ `255`. –Ø–∫—â–æ —á–∏—Å–ª–æ –Ω–µ –≤–º—ñ—â—É—î—Ç—å—Å—è –≤ —Ä–æ–∑–º—ñ—Ä —Ü—ñ–ª–æ—á–∏—Å–µ–ª—å–Ω–æ–≥–æ —Ç–∏–ø—É –∫–æ–Ω—Å—Ç–∞–Ω—Ç–∏ —á–∏ –∑–º—ñ–Ω–Ω–æ—ó, –ø—ñ–¥ —á–∞—Å –∫–æ–º–ø—ñ–ª—è—Ü—ñ—ó –∫–æ–¥—É –±—É–¥–µ –ø–æ–≤—ñ–¥–æ–º–ª–µ–Ω–æ –ø—Ä–æ –ø–æ–º–∏–ª–∫—É:

```swift
let cannotBeNegative: UInt8 = -1
// UInt8 –Ω–µ –º–æ–∂–µ –∑–±–µ—Ä—ñ–≥–∞—Ç–∏ –≤—ñ–¥'—î–º–Ω—ñ —á–∏—Å–ª–∞,
// —ñ —Ç–æ–º—É —Ü–µ –ø—Ä–∏–∑–≤–µ–¥–µ –¥–æ –ø–æ–º–∏–ª–∫–∏ –∫–æ–º–ø—ñ–ª—è—Ü—ñ—ó

let tooBig: Int8 = Int8.max + 1
// Int8 –Ω–µ –º–æ–∂–µ –∑–±–µ—Ä—ñ–≥–∞—Ç–∏ —á–∏—Å–ª–æ, –±—ñ–ª—å—à–µ –∑–∞ –º–∞–∫—Å–∏–º–∞–ª—å–Ω–µ –∑–Ω–∞—á–µ–Ω–Ω—è,
// —ñ —Ç–æ–º—É —Ü–µ –ø—Ä–∏–∑–≤–µ–¥–µ –¥–æ –ø–æ–º–∏–ª–∫–∏ –∫–æ–º–ø—ñ–ª—è—Ü—ñ—ó
```

–û—Å–∫—ñ–ª—å–∫–∏ –∫–æ–∂–µ–Ω —á–∏—Å–ª–æ–≤–∏–π —Ç–∏–ø –º–æ–∂–µ –∑–±–µ—Ä—ñ–≥–∞—Ç–∏ —Ä—ñ–∑–Ω–∏–π –¥—ñ–∞–ø–∞–∑–æ–Ω –∑–Ω–∞—á–µ–Ω—å, —Å–ª—ñ–¥ —Ä–æ–±–∏—Ç–∏ –ø–µ—Ä–µ—Ç–≤–æ—Ä–µ–Ω–Ω—è —á–∏—Å–ª–æ–≤–∏—Ö —Ç–∏–ø—ñ–≤ —É –∫–æ–∂–Ω–æ–º—É –≤–∏–ø–∞–¥–∫—É –æ–∫—Ä–µ–º–æ. –¶–µ–π –ø—ñ–¥—Ö—ñ–¥ –∑–∞—Ö–∏—â–∞—î –≤—ñ–¥ –ø–æ–º–∏–ª–æ–∫ –ø—Ä–∏—Ö–æ–≤–∞–Ω–∏—Ö –ø–µ—Ä–µ—Ç–≤–æ—Ä–µ–Ω—å, —ñ –¥–æ–ø–æ–º–∞–≥–∞—î —Ä–æ–±–∏—Ç–∏ –Ω–∞–º—ñ—Ä–∏ –ø–µ—Ä–µ—Ç–≤–æ—Ä–µ–Ω–Ω—è —Ç–∏–ø—ñ–≤ —É –∫–æ–¥—ñ —è–≤–Ω–∏–º–∏.

–©–æ–± –ø–µ—Ä–µ—Ç–≤–æ—Ä–∏—Ç–∏ –æ–¥–Ω–∏–º –∫–æ–Ω–∫—Ä–µ—Ç–Ω–∏–π —á–∏—Å–ª–æ–≤–∏–π —Ç–∏–ø –Ω–∞ —ñ–Ω—à–∏–π, —Å–ª—ñ–¥ –ø—Ä–æ—ñ–Ω—ñ—Ü—ñ–∞–ª—ñ–∑—É–≤–∞—Ç–∏ –Ω–æ–≤–µ —á–∏—Å–ª–æ –ø–æ—Ç—Ä—ñ–±–Ω–æ–≥–æ —Ç–∏–ø—É —ñ—Å–Ω—É—é—á–∏–º –∑–Ω–∞—á–µ–Ω–Ω—è–º. –£ –Ω–∞—Å—Ç—É–ø–Ω–æ–º—É –ø—Ä–∏–∫–ª–∞–¥—ñ, –∫–æ–Ω—Å—Ç–∞–Ω—Ç–∞ `twoThousand` –º–∞—î —Ç–∏–ø `UInt16`, —Ç–æ–¥—ñ —è–∫ –∫–æ–Ω—Å—Ç–∞–Ω—Ç–∞ `one` –º–∞—î —Ç–∏–ø `UInt8`. –á—Ö –Ω–µ –º–æ–∂–Ω–∞ –¥–æ–¥–∞–≤–∞—Ç–∏ –ø—Ä—è–º–æ, –±–æ –≤–æ–Ω–∏ —Ä—ñ–∑–Ω–∏—Ö —Ç–∏–ø—ñ–≤. –ó–∞–º—ñ—Å—Ç—å —Ü—å–æ–≥–æ, —É –ø—Ä–∏–∫–ª–∞–¥—ñ –≤–∏–∫–ª–∏–∫–∞—î—Ç—å—Å—è `UInt16(one)` —â–æ–± —Å—Ç–≤–æ—Ä–∏—Ç–∏ –Ω–æ–≤–µ —á–∏—Å–ª–æ —Ç–∏–ø—É `UInt16`, –ø—Ä–æ—ñ–Ω—ñ—Ü—ñ–∞–ª—ñ–∑–æ–≤–∞–Ω–µ –∑–Ω–∞—á–µ–Ω–Ω—è–º `one`, —ñ —Ü–µ –∑–Ω–∞—á–µ–Ω–Ω—è –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É—î—Ç—å—Å—è –∑–∞–º—ñ—Å—Ç—å –ø–æ—á–∞—Ç–∫–æ–≤–æ–≥–æ:

```swift
let twoThousand: UInt16 = 2_000
let one: UInt8 = 1
let twoThousandAndOne = twoThousand + UInt16(one)
```

–û—Å–∫—ñ–ª—å–∫–∏ –æ–±–∏–¥–≤–∞ –¥–æ–¥–∞–Ω–∫–∏ —Ç–µ–ø–µ—Ä –º–∞—é—Ç—å —Ç–∏–ø `UInt16`, –¥–æ–¥–∞–≤–∞–Ω–Ω—è –¥–æ–∑–≤–æ–ª—è—î—Ç—å—Å—è. –¢–∏–ø –≤–∏—Ö—ñ–¥–Ω–æ—ó –∫–æ–Ω—Å—Ç–∞–Ω—Ç–∏ \(`twoThousandAndOne`\) –≤–∏–∑–Ω–∞—á–∞—î—Ç—å—Å—è —è–∫ `UInt16`, –±–æ —Ü–µ —Å—É–º–∏ –¥–≤–æ—Ö –∑–Ω–∞—á–µ–Ω—å —Ç–∏–ø—É `UInt16`.

–í–∏—Ä–∞–∑ –≤–∏–¥—É `–Ø–∫–∏–π—Å—å–¢–∏–ø(–∑–Ø–∫–∏–º–æ—Å—å–ü–æ—á–∞—Ç–∫–æ–≤–∏–º–ó–Ω–∞—á–µ–Ω–Ω—è–º)` —î –∑–≤–∏—á–∞–π–Ω–∏–º —Å–ø–æ—Å–æ–±–æ–º –≤–∏–∫–ª–∏–∫–∞—Ç–∏ —ñ–Ω—ñ—Ü—ñ–∞–ª—ñ–∑–∞—Ç–æ—Ä —Ç–∏–ø—É —É Swift, —ñ –ø–µ—Ä–µ–¥–∞—Ç–∏ –π–æ–º—É –ø–æ—á–∞—Ç–∫–æ–≤–µ –∑–Ω–∞—á–µ–Ω–Ω—è. –ó–∞ –ª–∞—à—Ç—É–Ω–∫–∞–º–∏, —Ç–∏–ø `UInt16` –º–∞—î —ñ–Ω—ñ—Ü—ñ–∞–ª—ñ–∑–∞—Ç–æ—Ä, —â–æ –ø—Ä–∏–π–º–∞—î –∑–Ω–∞—á–µ–Ω–Ω—è —Ç–∏–ø—É `UInt8`, —ñ —Ç–æ–º—É —Ü–µ–π —ñ–Ω—ñ—Ü—ñ–∞–ª—ñ–∑–∞—Ç–æ—Ä –≤–∂–∏–≤–∞—î—Ç—å—Å—è –¥–ª—è —Å—Ç–≤–æ—Ä–µ–Ω–Ω—è –Ω–æ–≤–æ–≥–æ –∑–Ω–∞—á–µ–Ω–Ω—è —Ç–∏–ø—É `UInt16` –∑ —ñ—Å–Ω—É—é—á–æ–≥–æ –∑–Ω–∞—á–µ–Ω–Ω—è —Ç–∏–ø—É `UInt8`. –°—é–¥–∏ –Ω–µ –º–æ–∂–Ω–∞ –ø–µ—Ä–µ–¥–∞—Ç–∏ –±—É–¥—å-—è–∫–µ –∑–Ω–∞—á–µ–Ω–Ω—è, —Ü–µ –º–∞—î –±—É—Ç–∏ —Ç–∞–∫–∏–π —Ç–∏–ø, –¥–ª—è —è–∫–æ–≥–æ —É `UInt16` —î —ñ–Ω—ñ—Ü—ñ–∞–ª—ñ–∑–∞—Ç–æ—Ä. –û–¥–Ω–∞–∫ —É –º–æ–≤—ñ Swift —î –º–æ–∂–ª–∏–≤—ñ—Å—Ç—å —Ä–æ–∑—à–∏—Ä–∏—Ç–∏ —ñ—Å–Ω—É—é—á–∏–π —Ç–∏–ø, –¥–æ–¥–∞–≤—à–∏ –¥–æ –Ω—å–æ–≥–æ –Ω–æ–≤–∏–π —ñ–Ω—ñ—Ü—ñ–∞–ª—ñ–∑–∞—Ç–æ—Ä, —â–æ –ø—Ä–∏–π–º–∞—î –Ω–æ–≤–∏–π —Ç–∏–ø \(–≤–∫–ª—é—á–∞—é—á–∏ –∫–æ—Ä–∏—Å—Ç—É–≤–∞—Ü—å–∫–∏–π —Ç–∏–ø\). –î–µ—Ç–∞–ª—å–Ω—ñ—à–µ –ø—Ä–æ —Ü–µ –º–æ–∂–Ω–∞ –ø—Ä–æ—á–∏—Ç–∞—Ç–∏ —É —Ä–æ–∑–¥—ñ–ª—ñ [–†–æ–∑—à–∏—Ä–µ–Ω–Ω—è]({% link _book/1_language_guide/19_extensions.md %}).

### –ü–µ—Ä–µ—Ç–≤–æ—Ä–µ–Ω–Ω—è —Ü—ñ–ª–∏—Ö —á–∏—Å–µ–ª —Ç–∞ —á–∏—Å–µ–ª –∑ —Ä—É—Ö–æ–º–æ—é –∫–æ–º–æ—é

–ü–µ—Ä–µ—Ç–≤–æ—Ä–µ–Ω–Ω—è –º—ñ–∂ —Ü—ñ–ª–æ—á–∏—Å–µ–ª—å–Ω–∏–º–∏ —á–∏—Å–ª–æ–≤–∏–º–∏ —Ç–∏–ø–∞–º–∏ —Ç–∞ —Ç–∏–ø–∞–º–∏ –∑ —Ä—É—Ö–æ–º–æ—é –∫–æ–º–æ—é –º–∞—î –≤—ñ–¥–±—É–≤–∞—Ç–∏—Å—å —è–≤–Ω–æ:

```swift
let three = 3
let pointOneFourOneFiveNine = 0.14159
let pi = Double(three) + pointOneFourOneFiveNine
// –∫–æ–Ω—Å—Ç–∞–Ω—Ç–∞ pi –¥–æ—Ä—ñ–≤–Ω—é—î 3.14159, —ñ —ó—ó —Ç–∏–ø –≤–∏–∑–Ω–∞—á–µ–Ω–æ —è–∫ Double
```

–¢—É—Ç –∑–Ω–∞—á–µ–Ω–Ω—è –∫–æ–Ω—Å—Ç–∞–Ω—Ç–∏ `three` –≤–∏–∫–æ—Ä–∏—Å—Ç–∞–Ω–æ –¥–ª—è —Ç–æ–≥–æ, —â–æ–± —Å—Ç–≤–æ—Ä–∏—Ç–∏ –Ω–æ–≤–µ –∑–Ω–∞—á–µ–Ω–Ω—è —Ç–∏–ø—É `Double`, —ñ —Ç–æ–º—É –æ–±–∏–¥–≤–∞ –¥–æ–¥–∞–Ω–∫–∏ –º–∞—é—Ç—å –æ–¥–Ω–∞–∫–æ–≤–∏–π —Ç–∏–ø. –Ø–∫–±–∏ —Ü–µ –ø–µ—Ä–µ—Ç–≤–æ—Ä–µ–Ω–Ω—è –Ω–µ –º–∞–ª–æ –º—ñ—Å—Ü—è, –¥–æ–¥–∞–≤–∞–Ω–Ω—è –Ω–µ –±—É–ª–æ –± –¥–æ–∑–≤–æ–ª–µ–Ω–æ.

–ü–µ—Ä–µ—Ç–≤–æ—Ä–µ–Ω–Ω—è —á–∏—Å–µ–ª –∑ —Ä—É—Ö–æ–º–æ—é –∫–æ–º–æ—é –Ω–∞ —Ü—ñ–ª—ñ —Ç–∞–∫–æ–∂ –º–∞—î –±—É—Ç–∏ —è–≤–Ω–∏–º. –¶—ñ–ª–µ —á–∏—Å–ª–æ –º–æ–∂–Ω–∞ –ø—Ä–æ—ñ–Ω—ñ—Ü—ñ–∞–ª—ñ–∑—É–≤–∞—Ç–∏ –∑–Ω–∞—á–µ–Ω–Ω—è–º —Ç–∏–ø—É `Double` —á–∏ `Float`:

```swift
let integerPi = Int(pi)
// –∫–æ–Ω—Å—Ç–∞–Ω—Ç–∞ integerPi –¥–æ—Ä—ñ–≤–Ω—é—î 3, —ñ —ó—ó —Ç–∏–ø –≤–∏–∑–Ω–∞—á–µ–Ω–æ —è–∫ Int
```

–ó–Ω–∞—á–µ–Ω–Ω—è –∑ —Ä—É—Ö–æ–º–æ—é –∫–æ–º–æ—é –∑–∞–≤–∂–¥–∏ —É—Å—ñ–∫–∞—é—Ç—å—Å—è –ø—ñ–¥ —á–∞—Å —ñ–Ω—ñ—Ü—ñ–∞–ª—ñ–∑–∞—Ü—ñ—ó —Ü—ñ–ª–∏—Ö –∑–Ω–∞—á–µ–Ω—å —Ç–∞–∫–∏–º —á–∏–Ω–æ–º. –¢–æ–±—Ç–æ `4.75` —Å—Ç–∞–Ω–µ `4`, —ñ `-3.9` —Å—Ç–∞–Ω–µ `-3`.

> **–ü—Ä–∏–º—ñ—Ç–∫–∞**
>
> –ü—Ä–∞–≤–∏–ª–∞ –ø–æ—î–¥–Ω–∞–Ω–Ω—è —á–∏—Å–ª–æ–≤–∏—Ö –∫–æ–Ω—Å—Ç–∞–Ω—Ç —ñ –∑–º—ñ–Ω–Ω–∏—Ö –≤—ñ–¥—Ä—ñ–∑–Ω—è—é—Ç—å—Å—è –≤—ñ–¥ –ø—Ä–∞–≤–∏–ª –¥–ª—è —á–∏—Å–ª–æ–≤–∏—Ö –ª—ñ—Ç–µ—Ä–∞–ª—ñ–≤. –õ—ñ—Ç–µ—Ä–∞–ª—å–Ω–µ –∑–Ω–∞—á–µ–Ω–Ω—è `3` –º–æ–∂–µ –±—É—Ç–∏ –¥–æ–¥–∞–Ω–µ –ø—Ä—è–º–æ –¥–æ –ª—ñ—Ç–µ—Ä–∞–ª—å–Ω–æ–≥–æ –∑–Ω–∞—á–µ–Ω–Ω—è `0.14159`, –±–æ —á–∏—Å–ª–æ–≤—ñ –ª—ñ—Ç–µ—Ä–∞–ª–∏ –Ω–µ –º–∞—é—Ç—å —è–≤–Ω–æ–≥–æ —Ç–∏–ø—É —Å–∞–º—ñ –ø–æ —Å–æ–±—ñ. –á—Ö —Ç–∏–ø –≤–∏–∑–Ω–∞—á–∞—î—Ç—å—Å—è —Ç—ñ–ª—å–∫–∏ –≤ –º–æ–º–µ–Ω—Ç, –∫–æ–ª–∏ –≤–æ–Ω–∏ –æ—Ü—ñ–Ω—é—é—Ç—å—Å—è –∫–æ–º–ø—ñ–ª—è—Ç–æ—Ä–æ–º.

## –ü—Å–µ–≤–¥–æ–Ω—ñ–º–∏ —Ç–∏–ø—ñ–≤

_–ü—Å–µ–≤–¥–æ–Ω—ñ–º–∏ —Ç–∏–ø—ñ–≤_ –≤–∏–∑–Ω–∞—á–∞—é—Ç—å –∞–ª—å—Ç–µ—Ä–Ω–∞—Ç–∏–≤–Ω–µ —ñ–º'—è –¥–ª—è —ñ—Å–Ω—É—é—á–æ–≥–æ —Ç–∏–ø—É. –ü—Å–µ–≤–¥–æ–Ω—ñ–º–∏ —Ç–∏–ø—ñ–≤ –≤–∏–∑–Ω–∞—á–∞—é—Ç—å—Å—è –∑–∞ –¥–æ–ø–æ–º–æ–≥–æ—é –∫–ª—é—á–æ–≤–æ–≥–æ —Å–ª–æ–≤–∞ `typealias`.

–ü—Å–µ–≤–¥–æ–Ω—ñ–º–∏ —Ç–∏–ø—ñ–≤ –∫–æ—Ä–∏—Å–Ω—ñ, –∫–æ–ª–∏ –ø–æ—Ç—Ä—ñ–±–Ω–æ –ø–æ—Å–∏–ª–∞—Ç–∏—Å—å –Ω–∞ —ñ—Å–Ω—É—é—á–∏–π —Ç–∏–ø –∑–∞ —ñ–º–µ–Ω–µ–º, —â–æ –±—ñ–ª—å—à –¥–æ—Ä–µ—á–Ω–µ –≤ –¥–∞–Ω–æ–º—É –∫–æ–Ω—Ç–µ–∫—Å—Ç—ñ. –ù–∞–ø—Ä–∏–∫–ª–∞–¥, –ø—Ä–∏ —Ä–æ–±–æ—Ç—ñ –∑ —ñ–Ω—Ñ–æ—Ä–º–∞—Ü—ñ—î—é —Ñ—ñ–∫—Å–æ–≤–∞–Ω–æ–≥–æ —Ä–æ–∑–º—ñ—Ä—É —ñ–∑ –∑–æ–≤–Ω—ñ—à–Ω—å–æ–≥–æ –¥–∂–µ—Ä–µ–ª–∞:

```swift

```

–Ø–∫ —Ç—ñ–ª—å–∫–∏ –±—É–ª–æ –æ–≥–æ–ª–æ—à–µ–Ω–æ –ø—Å–µ–≤–¥–æ–Ω—ñ–º —Ç–∏–ø—É, –π–æ–≥–æ –º–æ–∂–Ω–∞ –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É–≤–∞—Ç–∏ –±—É–¥—å-–¥–µ –∑–∞–º—ñ—Å—Ç—å –ø–æ—á–∞—Ç–∫–æ–≤–æ–≥–æ —ñ–º–µ–Ω—ñ:

```swift
var maxAmplitudeFound = AudioSample.min
// maxAmplitudeFound —Ç–µ–ø–µ—Ä –¥–æ—Ä—ñ–≤–Ω—é—î 0
```

–¢—É—Ç `AudioSample` –≤–∏–∑–Ω–∞—á–µ–Ω–æ —è–∫ –ø—Å–µ–≤–¥–æ–Ω—ñ–º –¥–æ —Ç–∏–ø—É `UInt16`. –û—Å–∫—ñ–ª—å–∫–∏ —Ü–µ –ø—Å–µ–≤–¥–æ–Ω—ñ–º, –≤–∏–∫–ª–∏–∫ `AudioSample.min` –Ω–∞—Å–ø—Ä–∞–≤–¥—ñ –≤–∏–∫–ª–∏–∫–∞—î `UInt16.min`, –∫–æ—Ç—Ä–∏–π –Ω–∞–¥–∞—î –ø–æ—á–∞—Ç–∫–æ–≤–µ –∑–Ω–∞—á–µ–Ω–Ω—è `0` –∑–º—ñ–Ω–Ω—ñ–π `maxAmplitudeFound`.

## –ë—É–ª–µ–≤–∏–π —Ç–∏–ø –¥–∞–Ω–∏—Ö

–ú–æ–≤–∞ Swift –º–∞—î –±–∞–∑–æ–≤–∏–π _–±—É–ª–µ–≤–∏–π_ —Ç–∏–ø, —â–æ –Ω–∞–∑–∏–≤–∞—î—Ç—å—Å—è `Bool`. –ë—É–ª–µ–≤—ñ –∑–Ω–∞—á–µ–Ω–Ω—è —â–µ –Ω–∞–∑–∏–≤–∞—é—Ç—å _–ª–æ–≥—ñ—á–Ω–∏–º–∏_, –±–æ –≤–æ–Ω–∏ –º–æ–∂—É—Ç—å –±—É—Ç–∏ –ª–∏—à–µ —ñ—Å—Ç–∏–Ω–æ—é —á–∏ —Ö–∏–±–Ω–æ—é. –ú–æ–≤–∞ Swift –Ω–∞–¥–∞—î –¥–≤–∞ –±—É–ª–µ–≤–∏—Ö –∫–æ–Ω—Å—Ç–∞–Ω—Ç–Ω–∏—Ö –∑–Ω–∞—á–µ–Ω–Ω—è, `true` \(—ñ—Å—Ç–∏–Ω–∞\) —Ç–∞ `false` \(—Ö–∏–±–∞\):

```swift
let orangesAreOrange = true
let turnipsAreDelicious = false
```

–¢–∏–ø–∏ –∫–æ–Ω—Å—Ç–∞–Ω—Ç `orangesAreOrange` —Ç–∞ `turnipsAreDelicious` –±—É–ª–æ –≤–∏–∑–Ω–∞—á–µ–Ω–æ —è–∫ `Bool` —á–µ—Ä–µ–∑ —Ç–µ, —â–æ —ó—Ö –±—É–ª–æ –ø—Ä–æ—ñ–Ω—ñ—Ü—ñ–∞–ª—ñ–∑–æ–≤–∞–Ω–æ –±—É–ª–µ–≤–∏–º–∏ –ª—ñ—Ç–µ—Ä–∞–ª–∞–º–∏. –¢–∞–∫ —Å–∞–º–æ —è–∫ —ñ –∑ —Ç–∏–ø–∞–º–∏ `Int` —Ç–∞ `Double` –≤–∏—â–µ, –Ω–µ –ø–æ—Ç—Ä—ñ–±–Ω–æ —è–≤–Ω–æ –≤–∫–∞–∑—É–≤–∞—Ç–∏ —Ç–∏–ø `Bool` —è–∫—â–æ —ó–º –ø—Ä–∏—Å–≤–æ—é—î—Ç—å—Å—è –∑–Ω–∞—á–µ–Ω–Ω—è `true` —á–∏ `false` –æ–¥—Ä–∞–∑—É –ø—ñ—Å–ª—è —Å—Ç–≤–æ—Ä–µ–Ω–Ω—è. –í–∏–∑–Ω–∞—á–µ–Ω–Ω—è —Ç–∏–ø—ñ–≤ –¥–æ–∑–≤–æ–ª—è—î —Ä–æ–±–∏—Ç–∏ –∫–æ–¥ –º–æ–≤–æ—é Swift –∫–æ—Ä–æ—Ç—à–∏–º —Ç–∞ —á–∏—Ç–∞–±–µ–ª—å–Ω–∏–º –ø—ñ–¥ —á–∞—Å —ñ–Ω—ñ—Ü—ñ–∞–ª—ñ–∑–∞—Ü—ñ—ó –∫–æ–Ω—Å—Ç–∞–Ω—Ç —á–∏ –∑–º—ñ–Ω–Ω–∏—Ö –∑–Ω–∞—á–µ–Ω–Ω—è–º–∏, —á–∏–π —Ç–∏–ø –≤—ñ–¥–æ–º–æ.

–ë—É–ª–µ–≤—ñ –∑–Ω–∞—á–µ–Ω–Ω—è –∑–æ–∫—Ä–µ–º–∞ –∫–æ—Ä–∏—Å–Ω—ñ –ø—ñ–¥ —á–∞—Å —Ä–æ–±–æ—Ç–∏ –∑ —É–º–æ–≤–Ω–∏–º–∏ —ñ–Ω—Å—Ç—Ä—É–∫—Ü—ñ—è–º–∏, —Ç–∞–∫–∏–º–∏ —è–∫ —ñ–Ω—Å—Ç—Ä—É–∫—Ü—ñ—è `if`:

```swift
if turnipsAreDelicious {
    print("Mmm, tasty turnips!")
} else {
    print("Eww, turnips are horrible.")
}
// –ù–∞–¥—Ä—É–∫—É—î "Eww, turnips are horrible."
```

–£–º–æ–≤–Ω—ñ —ñ–Ω—Å—Ç—Ä—É–∫—Ü—ñ—ó, —Ç–∞–∫—ñ —è–∫ —ñ–Ω—Å—Ç—Ä—É–∫—Ü—ñ—è `if` –æ–ø–∏—Å–∞–Ω—ñ –±—ñ–ª—å—à –¥–µ—Ç–∞–ª—å–Ω–æ —É —Ä–æ–∑–¥—ñ–ª—ñ [–ü–æ—Ç—ñ–∫ –∫–µ—Ä—É–≤–∞–Ω–Ω—è]({% link _book/1_language_guide/4_control_flow.md %}).

–¢–∏–ø–æ–±–µ–∑–ø–µ—á–Ω—ñ—Å—Ç—å –º–æ–≤–∏ Swift –∑–∞–ø–æ–±—ñ–≥–∞—î –ø—ñ–¥—Å—Ç–∞–Ω–æ–≤—Ü—ñ –Ω–µ –±—É–ª–µ–≤–∏—Ö –∑–Ω–∞—á–µ–Ω—å –∑–∞–º—ñ—Å—Ç—å –±—É–ª–µ–≤–∏—Ö. –ù–∞—Å—Ç—É–ø–Ω–∏–π –ø—Ä–∏–∫–ª–∞–¥ –ø—Ä–∏–∑–≤–µ–¥–µ –¥–æ –ø–æ–≤—ñ–¥–æ–º–ª–µ–Ω–Ω—è –ø—Ä–æ –ø–æ–º–∏–ª–∫—É –∫–æ–º–ø—ñ–ª—è—Ü—ñ—ó:

```swift

    // –¶–µ–π –ø—Ä–∏–∫–ª–∞–¥ –Ω–µ —Å–∫–æ–º–ø—ñ–ª—é—î—Ç—å—Å—è, —ñ –±—É–¥–µ –ø–æ–≤—ñ–¥–æ–º–ª–µ–Ω–æ –ø—Ä–æ –ø–æ–º–∏–ª–∫—É.
}
```

–û–¥–Ω–∞–∫, –∞–ª—å—Ç–µ—Ä–Ω–∞—Ç–∏–≤–Ω–∏–π –ø—Ä–∏–∫–ª–∞–¥ –Ω–∏–∂—á–µ —î –¥—ñ–π—Å–Ω–∏–º:

```swift
let i = 1
if i == 1 {
    // –¶–µ–π –ø—Ä–∏–∫–ª–∞–¥ —É—Å–ø—ñ—à–Ω–æ —Å–∫–æ–º–ø—ñ–ª—é—î—Ç—å—Å—è
}
```

–†–µ–∑—É–ª—å—Ç–∞—Ç –ø–æ—Ä—ñ–≤–Ω—è–Ω–Ω—è `i == 1` –º–∞—î —Ç–∏–ø `Bool`, —ñ —Ç–æ–º—É –¥—Ä—É–≥–∏–π –ø—Ä–∏–∫–ª–∞–¥ –ø—Ä–æ—Ö–æ–¥–∏—Ç—å –ø–µ—Ä–µ–≤—ñ—Ä–∫—É —Ç–∏–ø—É. –ü–æ—Ä—ñ–≤–Ω—è–Ω–Ω—è –Ω–∞ –∫—à—Ç–∞–ª—Ç `i == 1` –æ–ø–∏—Å–∞–Ω–æ —É —Ä–æ–∑–¥—ñ–ª—ñ [–ë–∞–∑–æ–≤—ñ –æ–ø–µ—Ä–∞—Ç–æ—Ä–∏]({% link _book/1_language_guide/1_base_operators.md %}).

–Ø–∫ —ñ –≤ —ñ–Ω—à–∏—Ö –ø—Ä–∏–∫–ª–∞–¥–∞—Ö —Ç–∏–ø–æ–±–µ–∑–ø–µ—á–Ω–æ—Å—Ç—ñ —É –º–æ–≤—ñ Swift, —Ü–µ–π –ø—ñ–¥—Ö—ñ–¥ –∑–∞–ø–æ–±—ñ–≥–∞—î –≤–∏–ø–∞–¥–∫–æ–≤–∏–º –ø–æ–º–∏–ª–∫–∞–º —ñ –≥–∞—Ä–∞–Ω—Ç—É—î, —â–æ –Ω–∞–º—ñ—Ä –≤ –æ–∫—Ä–µ–º—ñ–π —Å–µ–∫—Ü—ñ—ó –∫–æ–¥—É —î –∑–∞–≤–∂–¥–∏ –∑—Ä–æ–∑—É–º—ñ–ª–∏–º.

## –ö–æ—Ä—Ç–µ–∂—ñ

_–ö–æ—Ä—Ç–µ–∂—ñ_ –≥—Ä—É–ø—É—é—Ç—å –∫—ñ–ª—å–∫–∞ –∑–Ω–∞—á–µ–Ω—å –≤ —î–¥–∏–Ω–µ —Å–∫–ª–∞–¥–µ–Ω–µ –∑–Ω–∞—á–µ–Ω–Ω—è. –ó–Ω–∞—á–µ–Ω–Ω—è –≤—Å–µ—Ä–µ–¥–∏–Ω—ñ –∫–æ—Ä—Ç–µ–∂—É –º–æ–∂—É—Ç—å –±—É—Ç–∏ –±—É–¥—å-—è–∫–æ–≥–æ —Ç–∏–ø—É —ñ –Ω–µ –æ–±–æ–≤'—è–∑–∫–æ–≤–æ –ø–æ–≤–∏–Ω–Ω—ñ –º–∞—Ç–∏ –æ–¥–Ω–∞–∫–æ–≤–∏–π —Ç–∏–ø.

–£ –Ω–∞—Å—Ç—É–ø–Ω–æ–º—É –ø—Ä–∏–∫–ª–∞–¥—ñ, `(404, "Not Found")` —î –∫–æ—Ä—Ç–µ–∂–µ–º, —â–æ –æ–ø–∏—Å—É—î _–∫–æ–¥ —Å—Ç–∞–Ω—É_ HTTP. [–ö–æ–¥ —Å—Ç–∞–Ω—É HTTP](https://uk.wikipedia.org/wiki/–°–ø–∏—Å–æ–∫_–∫–æ–¥—ñ–≤_—Å—Ç–∞–Ω—É_HTTP) - —Ü–µ —Å–ø–µ—Ü—ñ–∞–ª—å–Ω–µ –∑–Ω–∞—á–µ–Ω–Ω—è, —â–æ –ø–æ–≤–µ—Ä—Ç–∞—î—Ç—å—Å—è –≤–µ–±-—Å–µ—Ä–≤–µ—Ä–æ–º –ø—ñ—Å–ª—è –∫–æ–∂–Ω–æ–≥–æ –∑–∞–ø–∏—Ç—É –≤–µ–±-—Å—Ç–æ—Ä—ñ–Ω–∫–∏. –ö–æ–¥ —Å—Ç–∞–Ω—É `404 Not Found` –ø–æ–≤–µ—Ä—Ç–∞—î—Ç—å—Å—è —Ç–æ–¥—ñ, –∫–æ–ª–∏ –∑–∞–ø–∏—Ç–∞–Ω–∞ —Å—Ç–æ—Ä—ñ–Ω–∫–∞ –Ω–µ —ñ—Å–Ω—É—î.

```swift
let http404Error = (404, "–ù–µ –∑–Ω–∞–π–¥–µ–Ω–æ")

// http404Error –º–∞—î —Ç–∏–ø (Int, String), —ñ –¥–æ—Ä—ñ–≤–Ω—é—î (404, "–ù–µ –∑–Ω–∞–π–¥–µ–Ω–æ")
```

–ö–æ—Ä—Ç–µ–∂ `(404, "Not Found")` –≥—Ä—É–ø—É—î —Ä–∞–∑–æ–º –∑–Ω–∞—á–µ–Ω–Ω—è —Ç–∏–ø—ñ–≤ `Int` —Ç–∞ `String` —â–æ–± –Ω–∞–¥–∞—Ç–∏ –∫–æ–¥—É —Å—Ç–∞–Ω—É HTTP –¥–≤–∞ –æ–∫—Ä–µ–º–∏—Ö –∑–Ω–∞—á–µ–Ω–Ω—è: —á–∏—Å–ª–æ —ñ –∑—Ä—É—á–Ω–∏–π –¥–ª—è —Å–ø—Ä–∏–π–Ω—è—Ç—Ç—è –ª—é–¥–∏–Ω–æ—é –æ–ø–∏—Å. –ô–æ–≥–æ –º–æ–∂–Ω–∞ –æ–ø–∏—Å–∞—Ç–∏ —è–∫ ‚Äú–∫–æ—Ä—Ç–µ–∂ —Ç–∏–ø—É `(Int, String)`‚Äù.

–ú–æ–∂–Ω–∞ —Å—Ç–≤–æ—Ä–∏—Ç–∏ –∫–æ—Ä—Ç–µ–∂—ñ –∑ –±—É–¥—å-—è–∫–æ—ó –ø–µ—Ä–µ—Å—Ç–∞–Ω–æ–≤–∫–∏ —Ç–∏–ø—ñ–≤, —ñ –≤–æ–Ω–∏ –º–æ–∂—É—Ç—å –º—ñ—Å—Ç–∏—Ç–∏ —Å–∫—ñ–ª—å–∫–∏ –∑–∞–≤–≥–æ–¥–Ω–æ —Ä—ñ–∑–Ω–∏—Ö —Ç–∏–ø—ñ–≤. –ù—ñ—â–æ –Ω–µ –∑–∞–≤–∞–∂–∞—î –º–∞—Ç–∏ –∫–æ—Ä—Ç–µ–∂ —Ç–∏–ø—É `(Int, Int, Int)` –∞–±–æ `(String, Bool)`, –∞–±–æ –±—É–¥—å-—è–∫—É —ñ–Ω—à—É –Ω–µ–æ–±—Ö—ñ–¥–Ω—É –ø–µ—Ä–µ—Å—Ç–∞–Ω–æ–≤–∫—É —Ç–∏–ø—ñ–≤.

–ú–æ–∂–Ω–∞ _—Ä–æ–∑–∫–ª–∞—Å—Ç–∏_ –≤–º—ñ—Å—Ç –∫–æ—Ä—Ç–µ–∂—É –Ω–∞ –æ–∫—Ä–µ–º—ñ –∫–æ–Ω—Å—Ç–∞–Ω—Ç–∏ —á–∏ –∑–º—ñ–Ω–Ω—ñ, –¥–æ —è–∫–∏—Ö –º–æ–∂–Ω–∞ –∑–≤–µ—Ä—Ç–∞—Ç–∏—Å—å —è–∫ –∑–≤–∏—á–∞–π–Ω–æ:

```swift

// –ù–∞–¥—Ä—É–∫—É—î "–ö–æ–¥ —Å—Ç–∞–Ω—É –¥–æ—Ä—ñ–≤–Ω—é—î 404"

```

–Ø–∫—â–æ –ø–æ—Ç—Ä—ñ–±–Ω–æ —Ç—ñ–ª—å–∫–∏ –¥–µ—è–∫—ñ –∑—ñ –∑–Ω–∞—á–µ–Ω—å –∫–æ—Ä—Ç–µ–∂—É, –º–æ–∂–Ω–∞ –ø—Ä–æ—ñ–≥–Ω–æ—Ä—É–≤–∞—Ç–∏ —á–∞—Å—Ç–∏–Ω–∏ –∫–æ—Ä—Ç–µ–∂—É –∑–∞ –¥–æ–ø–æ–º–æ–≥–æ—é —Å–∏–º–≤–æ–ª—É –ø—ñ–¥–∫—Ä–µ—Å–ª–µ–Ω–Ω—è \(`_`\) –ø—ñ–¥ —á–∞—Å –¥–µ–∫–æ–º–ø–æ–∑–∏—Ü—ñ—ó –∫–æ—Ä—Ç–µ–∂—É:

```swift

// –ù–∞–¥—Ä—É–∫—É—î "–ö–æ–¥ —Å—Ç–∞—Ç—É—Å—É –¥–æ—Ä—ñ–≤–Ω—é—î 404"
```

–Ü–Ω—à–∏–º —Å–ø–æ—Å–æ–±–æ–º –¥–æ—Å—Ç—É–ø—É –¥–æ –æ–∫—Ä–µ–º–∏—Ö –µ–ª–µ–º–µ–Ω—Ç—ñ–≤ –∫–æ—Ä—Ç–µ–∂—É —î —á–∏—Å–ª–æ–≤—ñ —ñ–Ω–¥–µ–∫—Å–∏, —è–∫—ñ –ø–æ—á–∏–Ω–∞—é—Ç—å—Å—è –∑ –Ω—É–ª—å–æ–≤–æ–≥–æ:

```swift

// –ù–∞–¥—Ä—É–∫—É—î "–ö–æ–¥ —Å—Ç–∞—Ç—É—Å—É –¥–æ—Ä—ñ–≤–Ω—é—î 404"

// –ù–∞–¥—Ä—É–∫—É—î "–ü–æ–≤—ñ–¥–æ–º–ª–µ–Ω–Ω—è —Å—Ç–∞—Ç—É—Å—É —î –ù–µ –∑–Ω–∞–π–¥–µ–Ω–æ"
```

–¢–∞–∫–æ–∂ –º–æ–∂–Ω–∞ —ñ–º–µ–Ω—É–≤–∞—Ç–∏ –æ–∫—Ä–µ–º—ñ –µ–ª–µ–º–µ–Ω—Ç–∏ –≤ –∫–æ—Ä—Ç–µ–∂—ñ –ø—ñ–¥ —á–∞—Å –π–æ–≥–æ –æ–≥–æ–ª–æ—à–µ–Ω–Ω—è:

```swift
// –ù–∞–¥—Ä—É–∫—É—î "–ü–æ–≤—ñ–¥–æ–º–ª–µ–Ω–Ω—è —Å—Ç–∞—Ç—É—Å—É —î OK"
```

> **–ü—Ä–∏–º—ñ—Ç–∫–∞**

> –ö–æ—Ä—Ç–µ–∂—ñ –∑—Ä—É—á–Ω–æ –≤–∂–∏–≤–∞—Ç–∏ –¥–ª—è —Ç–∏–º—á–∞—Å–æ–≤–∏—Ö –≥—Ä—É–ø –ø–æ–≤'—è–∑–∞–Ω–∏—Ö –∑–Ω–∞—á–µ–Ω—å. –í–æ–Ω–∏ –Ω–µ –ø—ñ–¥—Ö–æ–¥—è—Ç—å –¥–ª—è —Å—Ç–≤–æ—Ä–µ–Ω–Ω—è —Å–∫–ª–∞–¥–Ω–∏—Ö —Å—Ç—Ä—É–∫—Ç—É—Ä –¥–∞–Ω–∏—Ö. –Ø–∫—â–æ —Å—Ç—Ä—É–∫—Ç—É—Ä–∞ –¥–∞–Ω–∏—Ö –º–æ–∂–µ –∂–∏—Ç–∏ –ø–æ–∑–∞ —Ç–∏–º—á–∞—Å–æ–≤–∏–º –∫–æ–Ω—Ç–µ–∫—Å—Ç–æ–º, —Å–ª—ñ–¥ –º–æ–∂–µ–ª—é–≤–∞—Ç–∏ —ó—ó –∑–∞ –¥–æ–ø–æ–º–æ–≥–æ—é –∫–ª–∞—Å—É —á–∏ —Å—Ç—Ä—É–∫—Ç—É—Ä–∏, –∞ –Ω–µ –∫–æ—Ä—Ç–µ–∂—É. –ë—ñ–ª—å—à –¥–µ—Ç–∞–ª—å–Ω—É —ñ–Ω—Ñ–æ—Ä–º–∞—Ü—ñ—é –º–æ–∂–Ω–∞ –∑–Ω–∞–π—Ç–∏ —É —Ä–æ–∑–¥—ñ–ª—ñ [–ö–ª–∞—Å–∏ —Ç–∞ —Å—Ç—Ä—É–∫—Ç—É—Ä–∏]({% link _book/1_language_guide/8_classes_and_structures.md %}).

## –û–ø—Ü—ñ–æ–Ω–∞–ª–∏

_–û–ø—Ü—ñ–æ–Ω–∞–ª–∏_ —Å–ª—ñ–¥ –≤–∂–∏–≤–∞—Ç–∏, –∫–æ–ª–∏ –∑–Ω–∞—á–µ–Ω–Ω—è –º–æ–∂–µ –±—É—Ç–∏ –≤—ñ–¥—Å—É—Ç–Ω—î. –û–ø—Ü—ñ–æ–Ω–∞–ª–∏ –≤—ñ–¥–æ–±—Ä–∞–∂–∞—é—Ç—å –¥–≤—ñ –º–æ–∂–ª–∏–≤–æ—Å—Ç—ñ: –∞–±–æ _—ñ—Å–Ω—É—î_ –∑–Ω–∞—á–µ–Ω–Ω—è, —ñ –º–æ–∂–Ω–∞ –π–æ–≥–æ —Ä–æ–∑–≥–æ—Ä–Ω—É—Ç–∏ —â–æ–± –æ—Ç—Ä–∏–º–∞—Ç–∏ —Ü–µ –∑–Ω–∞—á–µ–Ω–Ω—è, –∞–±–æ _–Ω–µ —ñ—Å–Ω—É—î_ –∂–æ–¥–Ω–æ–≥–æ –∑–Ω–∞—á–µ–Ω–Ω—è.

> **–ü—Ä–∏–º—ñ—Ç–∫–∞**
>
> –ö–æ–Ω—Ü–µ–ø—Ü—ñ—è –æ–ø—Ü—ñ–æ–Ω–∞–ª—ñ–≤ –Ω–µ —ñ—Å–Ω—É—î —É –º–æ–≤–∞—Ö C —á–∏ Objective-C. –ù–∞–π–±–ª–∏–∂—á–∏–º –ø–æ–Ω—è—Ç—Ç—è–º —É Objective-C —î –º–æ–∂–ª–∏–≤—ñ—Å—Ç—å –ø–æ–≤–µ—Ä–Ω—É—Ç–∏ `nil` —ñ–∑ –º–µ—Ç–æ–¥—É, —â–æ –∑–∞–∑–≤–∏—á–∞–π –ø–æ–≤–µ—Ä—Ç–∞—î –æ–±'—î–∫—Ç, –≤ —Ç–∞–∫–æ–º—É –≤–∏–ø–∞–¥–∫—É `nil` –æ–∑–Ω–∞—á–∞—î "–≤—ñ–¥—Å—É—Ç–Ω—ñ—Å—Ç—å –≤–∞–ª—ñ–¥–Ω–æ–≥–æ –æ–±'—î–∫—Ç—É". –û–¥–Ω–∞–∫ —Ü–µ –ø—Ä–∞—Ü—é—î –ª–∏—à–µ –∑ –æ–±'—î–∫—Ç–∞–º–∏, —ñ –Ω–µ –ø—Ä–∞—Ü—é—î –Ω—ñ –∑ —Å—Ç—Ä—É–∫—Ç—É—Ä–∞–º–∏, –Ω—ñ –∑ –±–∞–∑–æ–≤–∏–º–∏ —Ç–∏–ø–∞–º–∏ –º–æ–≤–∏ C, –Ω—ñ –∑ –ø–µ—Ä–µ—á–∏—Å–ª–µ–Ω–Ω—è–º–∏. –î–ª—è —Ç–∞–∫–∏—Ö —Ç–∏–ø—ñ–≤ –º–µ—Ç–æ–¥–∏ Objective-C —è–∫ –ø—Ä–∞–≤–∏–ª–æ –ø–æ–≤–µ—Ä—Ç–∞—é—Ç—å —Å–ø–µ—Ü—ñ–∞–ª—å–Ω–µ –∑–Ω–∞—á–µ–Ω–Ω—è \(—Ç–∞–∫–µ —è–∫ `NSNotFound`\) —â–æ–± –ø–æ–∑–Ω–∞—á–∏—Ç–∏ –≤—ñ–¥—Å—É—Ç–Ω—ñ—Å—Ç—å –∑–Ω–∞—á–µ–Ω–Ω—è. –¶–µ–π –ø—ñ–¥—Ö—ñ–¥ –±–∞–∑—É—î—Ç—å—Å—è –Ω–∞ –ø—Ä–∏–ø—É—â–µ–Ω–Ω—ñ, —â–æ —Ç–æ–π, —Ö—Ç–æ –≤–∏–∫–ª–∏–∫–∞—î –º–µ—Ç–æ–¥, –∑–Ω–∞—î –ø—Ä–æ —Ü–µ —Å–ø–µ—Ü—ñ–∞–ª—å–Ω–µ –∑–Ω–∞—á–µ–Ω–Ω—è, —ñ –ø–∞–º'—è—Ç–∞—î, —â–æ —Å–ª—ñ–¥ –π–æ–≥–æ –ø–µ—Ä–µ–≤—ñ—Ä–∏—Ç–∏. –û–ø—Ü—ñ–æ–Ω–∞–ª–∏ —É –º–æ–≤—ñ Swift –¥–æ–∑–≤–æ–ª—è—é—Ç—å –≤–∫–∞–∑—É–≤–∞—Ç–∏ –Ω–∞ –≤—ñ–¥—Å—É—Ç–Ω—ñ—Å—Ç—å –∑–Ω–∞—á–µ–Ω–Ω—è _–≤–∑–∞–≥–∞–ª—ñ –¥–ª—è –±—É–¥—å-—è–∫–æ–≥–æ —Ç–∏–ø—É_, –±–µ–∑ –Ω–µ–æ–±—Ö—ñ–¥–Ω–æ—Å—Ç—ñ —É —Å–ø–µ—Ü—ñ–∞–ª—å–Ω–∏—Ö –∫–æ–Ω—Å—Ç–∞–Ω—Ç–∞—Ö.

–û—Å—å –ø—Ä–∏–∫–ª–∞–¥ —Ç–æ–≥–æ —è–∫ –æ–ø—Ü—ñ–æ–Ω–∞–ª–∏ –º–æ–∂—É—Ç—å –±—É—Ç–∏ –≤–∏–∫–æ—Ä–∏—Å—Ç–∞–Ω—ñ —â–æ–± –≤–ø–æ—Ä–∞—Ç–∏—Å—å —ñ–∑ –≤—ñ–¥—Å—É—Ç–Ω—ñ—Å—Ç—é –∑–Ω–∞—á–µ–Ω–Ω—è. –¢–∏–ø `Int` —É Swift –º–∞—î —ñ–Ω—ñ—Ü—ñ–∞–ª—ñ–∑–∞—Ç–æ—Ä, —â–æ –Ω–∞–º–∞–≥–∞—î—Ç—å—Å—è –∫–æ–Ω–≤–µ—Ä—Ç—É–≤–∞—Ç–∏ —Ä—è–¥–∫–æ–≤–µ –∑–Ω–∞—á–µ–Ω–Ω—è `String` —É –∑–Ω–∞—á–µ–Ω–Ω—è `Int`. –û–¥–Ω–∞–∫ –Ω–µ –∫–æ–∂–Ω–∏–π —Ä—è–¥–æ–∫ –º–æ–∂–Ω–∞ –∫–æ–Ω–≤–µ—Ä—Ç—É–≤–∞—Ç–∏ —É —Ü—ñ–ª–µ —á–∏—Å–ª–æ. –†—è–¥–æ–∫ `"123"` –º–æ–∂–Ω–∞ –∫–æ–Ω–≤–µ—Ä—Ç—É–≤–∞—Ç–∏ —É —á–∏—Å–ª–æ–≤–µ –∑–Ω–∞—á–µ–Ω–Ω—è `123`, –∞–ª–µ —Ä—è–¥–æ–∫ `"–ø—Ä–∏–≤—ñ—Ç, —Å–≤—ñ—Ç"` –Ω–µ –º–æ–∂–Ω–∞ –æ—á–µ–≤–∏–¥–Ω–∏–º —á–∏–Ω–æ–º –∫–æ–Ω–≤–µ—Ä—Ç—É–≤–∞—Ç–∏ —É —Ä—è–¥–æ–∫.

–£ –Ω–∞—Å—Ç—É–ø–Ω–æ–º—É –ø—Ä–∏–∫–ª–∞–¥—ñ —ñ–Ω—ñ—Ü—ñ–∞–ª—ñ–∑–∞—Ç–æ—Ä –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É—î—Ç—å—Å—è –¥–ª—è –∫–æ–Ω–≤–µ—Ä—Ç–∞—Ü—ñ—ó —Ä—è–¥–∫—É `String` —É —á–∏—Å–ª–æ `Int`:

–û—Å–∫—ñ–ª—å–∫–∏ —ñ–Ω—ñ—Ü—ñ–∞–ª—ñ–∑–∞—Ç–æ—Ä –º–æ–∂–µ –ø—Ä–æ–≤–∞–ª–∏—Ç–∏—Å—å, –≤—ñ–Ω –ø–æ–≤–µ—Ä—Ç–∞—î –æ–ø—Ü—ñ–æ–Ω–∞–ª—å–Ω–µ –∑–Ω–∞—á–µ–Ω–Ω—è `Int`, –∑–∞–º—ñ—Å—Ç—å –ø—Ä–æ—Å—Ç–æ–≥–æ `Int`. –û–ø—Ü—ñ–æ–Ω–∞–ª—å–Ω–∏–π —Ç–∏–ø `Int` –ø–∏—à–µ—Ç—å—Å—è —è–∫ `Int?`, –∞ –Ω–µ `Int`. –ó–Ω–∞–∫ –ø–∏—Ç–∞–Ω–Ω—è –ø–æ–∑–Ω–∞—á–∞—î —Ç–µ, —â–æ –∑–Ω–∞—á–µ–Ω–Ω—è –º—ñ—Å—Ç–∏—Ç—å –æ–ø—Ü—ñ–æ–Ω–∞–ª, —ñ –æ–∑–Ω–∞—á–∞—î, —â–æ –≤–æ–Ω–æ –º–æ–∂–µ –º—ñ—Å—Ç–∏—Ç–∏ _–¥–µ—è–∫–µ_ –∑–Ω–∞—á–µ–Ω–Ω—è `Int`, –∞–±–æ –º–æ–∂–µ –Ω–µ –º—ñ—Å—Ç–∏—Ç–∏ _–∂–æ–¥–Ω–æ–≥–æ –∑–Ω–∞—á–µ–Ω–Ω—è –≤–∑–∞–≥–∞–ª—ñ_. \(–í–æ–Ω–æ –Ω–µ –º–æ–∂–µ –º—ñ—Å—Ç–∏—Ç–∏ –±—É–¥—å-—â–æ —ñ–Ω—à–µ, –±—É–¥—å —Ç–æ –∑–Ω–∞—á–µ–Ω–Ω—è —Ç–∏–ø—É `Bool` —á–∏ `String`. –í–æ–Ω–æ –º–æ–∂–µ –º—ñ—Å—Ç–∏—Ç–∏ –∞–±–æ `Int`, –∞–±–æ –Ω—ñ—á–æ–≥–æ –≤–∑–∞–≥–∞–ª—ñ.\)

### nil

–©–æ–± –æ–ø—Ü—ñ–æ–Ω–∞–ª—å–Ω–∞ –∑–º—ñ–Ω–Ω–∞ –Ω–µ –º—ñ—Å—Ç–∏–ª–∞ –∂–æ–¥–Ω–æ–≥–æ –∑–Ω–∞—á–µ–Ω–Ω—è –≤–∑–∞–≥–∞–ª—ñ, —Å–ª—ñ–¥ –ø—Ä–∏—Å–≤–æ—ó—Ç–∏ —ó–π —Å–ø–µ—Ü—ñ–∞–ª—å–Ω–µ –∑–Ω–∞—á–µ–Ω–Ω—è `nil`:

### –Ü–Ω—Å—Ç—Ä—É–∫—Ü—ñ—è If —Ç–∞ –ü—Ä–∏–º—É—Å–æ–≤–µ —Ä–æ–∑–≥–æ—Ä—Ç–∞–Ω–Ω—è

```swift
// –ù–∞–¥—Ä—É–∫—É—î "convertedNumber –º—ñ—Å—Ç–∏—Ç—å –¥–µ—è–∫–µ —Ü—ñ–ª–æ—á–∏—Å–µ–ª—å–Ω–µ –∑–Ω–∞—á–µ–Ω–Ω—è."
```

–Ø–∫ —Ç—ñ–ª—å–∫–∏ –º–∏ –≤–ø–µ–≤–Ω–µ–Ω—ñ, —â–æ –æ–ø—Ü—ñ–æ–Ω–∞–ª _–º—ñ—Å—Ç–∏—Ç—å_ –∑–Ω–∞—á–µ–Ω–Ω—è, –º–æ–∂–Ω–∞ –æ—Ç—Ä–∏–º–∞—Ç–∏ —Ü–µ –∑–Ω–∞—á–µ–Ω–Ω—è, –¥–æ–¥–∞–≤—à–∏ –∑–Ω–∞–∫ –æ–∫–ª–∏–∫—É \(`!`\) –Ω–∞–ø—Ä–∏–∫—ñ–Ω—Ü—ñ —ñ–º–µ–Ω—ñ –æ–ø—Ü—ñ–æ–Ω–∞–ª—É. –ó–Ω–∞–∫ –æ–∫–ª–∏–∫—É —Ñ–∞–∫—Ç–∏—á–Ω–æ –≥–æ–≤–æ—Ä–∏—Ç—å "–Ø –∑–Ω–∞—é, —â–æ —Ü–µ–π –æ–ø—Ü—ñ–æ–Ω–∞–ª —Ç–æ—á–Ω–æ –º–∞—î –∑–Ω–∞—á–µ–Ω–Ω—è, –±—É–¥—å-–ª–∞—Å–∫–∞, –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É–π—Ç–µ –π–æ–≥–æ." –¶—è –æ–ø–µ—Ä–∞—Ü—ñ—è –≤—ñ–¥–æ–º–∞ —è–∫ _–ø—Ä–∏–º—É—Å–æ–≤–µ —Ä–æ–∑–≥–æ—Ä—Ç–∞–Ω–Ω—è_ –∑–Ω–∞—á–µ–Ω–Ω—è –æ–ø—Ü—ñ–æ–Ω–∞–ª—É:

```swift
if convertedNumber != nil {

print("convertedNumber –º–∞—î —Ü—ñ–ª–æ—á–∏—Å–µ–ª—å–Ω–µ –∑–Ω–∞—á–µ–Ω–Ω—è \(convertedNumber!).")

}

// Prints "convertedNumber –º–∞—î —Ü—ñ–ª–æ—á–∏—Å–µ–ª—å–Ω–µ –∑–Ω–∞—á–µ–Ω–Ω—è 123."
```

–ë—ñ–ª—å—à –¥–µ—Ç–∞–ª—å–Ω–æ –æ–∑–Ω–∞–π–æ–º–∏—Ç–∏—Å—å –∑ —ñ–Ω—Å—Ç—Ä—É–∫—Ü—ñ—î—é `if` –º–æ–∂–Ω–∞ –≤ —Ä–æ–∑–¥—ñ–ª—ñ [–ü–æ—Ç—ñ–∫ –∫–µ—Ä—É–≤–∞–Ω–Ω—è]({% link _book/1_language_guide/4_control_flow.md %}).

> **–ü—Ä–∏–º—ñ—Ç–∫–∞**
>
> –°–ø—Ä–æ–±–∞ –≤–∏–∫–æ—Ä–∏—Å—Ç–∞—Ç–∏ `!` –¥–ª—è –¥–æ—Å—Ç—É–ø—É –¥–æ –Ω–µ—ñ—Å–Ω—É—é—á–æ–≥–æ –∑–Ω–∞—á–µ–Ω–Ω—è –æ–ø—Ü—ñ–æ–Ω–∞–ª—É –ø—Ä–∏–∑–≤–µ–¥–µ –¥–æ –ø–æ–º–∏–ª–∫–∏ —á–∞—Å—É –≤–∏–∫–æ–Ω–∞–Ω–Ω—è. –ü–µ—Ä–µ–¥ —Ç–∏–º, —è–∫ –≤–∂–∏–≤–∞—Ç–∏ `!` –¥–ª—è –ø—Ä–∏–º—É—Å–æ–≤–æ–≥–æ —Ä–æ–∑–≥–æ—Ä—Ç–∞–Ω–Ω—è –∑–Ω–∞—á–µ–Ω–Ω—è –æ–ø—Ü—ñ–æ–Ω–∞–ª—É, —Å–ª—ñ–¥ —É–ø–µ–≤–Ω–∏—Ç–∏—Å—å, —â–æ –≤—ñ–Ω –º—ñ—Å—Ç–∏—Ç—å –Ω–µ `nil`.

### –û–ø—Ü—ñ–æ–Ω–∞–ª—å–Ω–µ –∑–≤'—è–∑—É–≤–∞–Ω–Ω—è

–û–ø—Ü—ñ–æ–Ω–∞–ª—å–Ω–µ –∑–≤'—è–∑—É–≤–∞–Ω–Ω—è –≤–∂–∏–≤–∞—î—Ç—å—Å—è —â–æ–± –¥—ñ–∑–Ω–∞—Ç–∏—Å—å, —á–∏ –º—ñ—Å—Ç–∏—Ç—å –æ–ø—Ü—ñ–æ–Ω–∞–ª –∑–Ω–∞—á–µ–Ω–Ω—è, —ñ —è–∫—â–æ –º—ñ—Å—Ç–∏—Ç—å, –∑—Ä–æ–±–∏—Ç–∏ —Ü–µ –∑–Ω–∞—á–µ–Ω–Ω—è –¥–æ—Å—Ç—É–ø–Ω–∏–º —è–∫ —Ç–∏–º—á–∞—Å–æ–≤–∞ –∫–æ–Ω—Å—Ç–∞–Ω—Ç–∞ —á–∏ –∑–º—ñ–Ω–Ω–∞. –û–ø—Ü—ñ–æ–Ω–∞–ª—å–Ω–µ –∑–≤'—è–∑—É–≤–∞–Ω–Ω—è –º–æ–∂–µ –≤–∂–∏–≤–∞—Ç–∏—Å—å –∑ —ñ–Ω—Å—Ç—Ä—É–∫—Ü—ñ—è–º–∏ `if` —Ç–∞ `while` —â–æ–± –ø–µ—Ä–µ–≤—ñ—Ä–∏—Ç–∏ –æ–ø—Ü—ñ–æ–Ω–∞–ª –Ω–∞ –Ω–∞—è–≤–Ω—ñ—Å—Ç—å –∑–Ω–∞—á–µ–Ω–Ω—è, –≤–∏—Ç—è–≥—Ç–∏ —Ü–µ –∑–Ω–∞—á–µ–Ω–Ω—è —É –∫–æ–Ω—Å—Ç–∞–Ω—Ç—É —á–∏ –∑–º—ñ–Ω–Ω—É, —ñ –≤—Å–µ —Ü–µ —É –≤–∏–≥–ª—è–¥—ñ —î–¥–∏–Ω–æ—ó –¥—ñ—ó. –Ü–Ω—Å—Ç—Ä—É–∫—Ü—ñ—ó `if` —Ç–∞ `while` –±—ñ–ª—å—à –¥–µ—Ç–∞–ª—å–Ω–æ –æ–ø–∏—Å–∞–Ω—ñ —É —Ä–æ–∑–¥—ñ–ª—ñ [–ü–æ—Ç—ñ–∫ –∫–µ—Ä—É–≤–∞–Ω–Ω—è]({% link _book/1_language_guide/4_control_flow.md %}).

–°–∏–Ω—Ç–∞–∫—Å–∏—Å –æ–ø—Ü—ñ–æ–Ω–∞–ª—å–Ω–æ–≥–æ –∑–≤'—è–∑—É–≤–∞–Ω–Ω—è –¥–ª—è —ñ–Ω—Å—Ç—Ä—É–∫—Ü—ñ—ó `if` –≤–∏–≥–ª—è–¥–∞—î –Ω–∞—Å—Ç—É–ø–Ω–∏–º —á–∏–Ω–æ–º:

```swift
if let <—ñ–º'—è–ö–æ–Ω—Å—Ç–∞–Ω—Ç–∏> = <—è–∫–∏–π—Å—å–û–ø—Ü—ñ–æ–Ω–∞–ª> {
    <—ñ–Ω—Å—Ç—Ä—É–∫—Ü—ñ—ó>
}
```

–ú–æ–∂–Ω–∞ –ø–µ—Ä–µ–ø–∏—Å–∞—Ç–∏ –ø—Ä–∏–∫–ª–∞–¥ –∑ `possibleNumber` —ñ–∑ —Å–µ–∫—Ü—ñ—ó [–û–ø—Ü—ñ–æ–Ω–∞–ª–∏]({% link _book/1_language_guide/0_the_basics.md %}#–æ–ø—Ü—ñ–æ–Ω–∞–ª–∏) –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É—é—á–∏ –æ–ø—Ü—ñ–æ–Ω–∞–ª—å–Ω–µ –∑–≤'—è–∑—É–≤–∞–Ω–Ω—è –∑–∞–º—ñ—Å—Ç—å –ø—Ä–∏–º—É—Å–æ–≤–æ–≥–æ —Ä–æ–∑–≥–æ—Ä—Ç–∞–Ω–Ω—è:

```swift
if let actualNumber = Int(possibleNumber) {
    print("\"\(possibleNumber)\" –º–∞—î —Ü—ñ–ª–æ—á–∏—Å–µ–ª—å–Ω–µ –∑–Ω–∞—á–µ–Ω–Ω—è \(actualNumber)")
} else {
    print("\"\(possibleNumber)\" –Ω–µ –º–æ–∂–Ω–∞ –∫–æ–Ω–≤–µ—Ä—Ç—É–≤–∞—Ç–∏ —É —Ü—ñ–ª–µ —á–∏—Å–ª–æ")
}
// –ù–∞–¥—Ä—É–∫—É—î ""123" –º–∞—î —Ü—ñ–ª–æ—á–∏—Å–µ–ª—å–Ω–µ –∑–Ω–∞—á–µ–Ω–Ω—è 123"
```

–¶–µ–π –∫–æ–¥ –º–æ–∂–Ω–∞ –ø—Ä–æ—á–∏—Ç–∞—Ç–∏ —Ç–∞–∫:

‚Äú–Ø–∫—â–æ –æ–ø—Ü—ñ–æ–Ω–∞–ª—å–Ω–∏–π `Int`, —â–æ –ø–æ–≤–µ—Ä—Ç–∞—î—Ç—å—Å—è –∑ `Int(possibleNumber)`, –º—ñ—Å—Ç–∏—Ç—å –∑–Ω–∞—á–µ–Ω–Ω—è, —Å—Ç–≤–æ—Ä–∏—Ç–∏ –Ω–æ–≤—É –∫–æ–Ω—Å—Ç–∞–Ω—Ç—É –∑ –Ω–∞–∑–≤–æ—é `actualNumber` —ñ —ñ–Ω—ñ—Ü—ñ–∞–ª—ñ–∑—É–≤–∞—Ç–∏ —ó—ó –∑–Ω–∞—á–µ–Ω–Ω—è–º, —â–æ –º—ñ—Å—Ç–∏—Ç—å—Å—è –≤ –æ–ø—Ü—ñ–æ–Ω–∞–ª—ñ.‚Äù

–Ø–∫—â–æ –ø–µ—Ä–µ—Ç–≤–æ—Ä–µ–Ω–Ω—è —É—Å–ø—ñ—à–Ω–µ, –∫–æ–Ω—Å—Ç–∞–Ω—Ç–∞ `actualNumber` —Å—Ç–∞—î –¥–æ—Å—Ç—É–ø–Ω–æ—é –¥–ª—è –≤–∏–∫–æ—Ä–∏—Å—Ç–∞–Ω–Ω—è –≤—Å–µ—Ä–µ–¥–∏–Ω—ñ –ø–µ—Ä—à–æ—ó –≥—ñ–ª–∫–∏ —ñ–Ω—Å—Ç—Ä—É–∫—Ü—ñ—ó `if`. –á—ó –≤–∂–µ –±—É–ª–æ –ø—Ä–æ—ñ–Ω—ñ—Ü—ñ–∞–ª—ñ–∑–æ–≤–∞–Ω–æ –∑–Ω–∞—á–µ–Ω–Ω—è–º, —â–æ –º—ñ—Å—Ç–∏–ª–æ—Å—å _–≤—Å–µ—Ä–µ–¥–∏–Ω—ñ_ –æ–ø—Ü—ñ–æ–Ω–∞–ª—É, —ñ —Ç–æ–º—É –Ω–µ–ø–æ—Ç—Ä—ñ–±–Ω–æ –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É–≤–∞—Ç–∏ —Å—É—Ñ—ñ–∫—Å `!` –¥–ª—è –¥–æ—Å—Ç—É–ø—É –¥–æ –∑–Ω–∞—á–µ–Ω–Ω—è. –£ —Ü—å–æ–º—É –ø—Ä–∏–∫–ª–∞–¥—ñ `actualNumber` –ø—Ä–æ—Å—Ç–æ –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É—î—Ç—å—Å—è –¥–ª—è –¥—Ä—É–∫—É —Ä–µ–∑—É–ª—å—Ç–∞—Ç—É –ø–µ—Ä–µ—Ç–≤–æ—Ä–µ–Ω–Ω—è.

–ó –æ–ø—Ü—ñ–æ–Ω–∞–ª—å–Ω–∏–º –∑–≤'—è–∑—É–≤–∞–Ω–Ω—è–º –º–æ–∂–Ω–∞ –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É–≤–∞—Ç–∏ —è–∫ –∫–æ–Ω—Å—Ç–∞–Ω—Ç–∏, —Ç–∞–∫ —ñ –∑–º—ñ–Ω–Ω—ñ. –Ø–∫—â–æ –Ω–µ–æ–±—Ö—ñ–¥–Ω–æ –º–∞–Ω—ñ–ø—É–ª—é–≤–∞—Ç–∏ –∑–Ω–∞—á–µ–Ω–Ω—è–º `actualNumber` –≤—Å–µ—Ä–µ–¥–∏–Ω—ñ –ø–µ—Ä—à–æ—ó –≥—ñ–ª–∫–∏ —ñ–Ω—Å—Ç—Ä—É–∫—Ü—ñ—ó `if`, –º–æ–∂–Ω–∞ –ø—Ä–æ—Å—Ç–æ –∑–∞–º—ñ—Å—Ç—å `if let actualNumber` –Ω–∞–ø–∏—Å–∞—Ç–∏ `if var actualNumber`, —ñ –∑–Ω–∞—á–µ–Ω–Ω—è, —â–æ –º—ñ—Å—Ç–∏—Ç—å—Å—è –≤—Å–µ—Ä–µ–¥–∏–Ω—ñ –æ–ø—Ü—ñ–æ–Ω–∞–ª—É —Å—Ç–∞–Ω–µ –¥–æ—Å—Ç—É–ø–Ω–æ—é —É –≤–∏–≥–ª—è–¥—ñ –∑–º—ñ–Ω–Ω–æ—ó –∑–∞–º—ñ—Å—Ç—å –∫–æ–Ω—Å—Ç–∞–Ω—Ç–∏.

–í —î–¥–∏–Ω—É —ñ–Ω—Å—Ç—Ä—É–∫—Ü—ñ—é `if` –º–æ–∂–Ω–∞ –≤–∫–ª—é—á–∏—Ç–∏ —Å—Ç—ñ–ª—å–∫–∏ –æ–ø—Ü—ñ–æ–Ω–∞–ª—å–Ω–∏—Ö –∑–≤'—è–∑—É–≤–∞–Ω—å —ñ –±—É–ª–µ–≤–∏—Ö —É–º–æ–≤, —Å–∫—ñ–ª—å–∫–∏ –ø–æ—Ç—Ä—ñ–±–Ω–æ, —Ä–æ–∑–¥—ñ–ª–∏–≤—à–∏ —ó—Ö –∫–æ–º–∞–º–∏. –Ø–∫—â–æ –±—É–¥—å-—è–∫–µ –∑—ñ –∑–Ω–∞—á–µ–Ω—å –≤ –æ–ø—Ü—ñ–æ–Ω–∞–ª—å–Ω–∏—Ö –∑–≤'—è–∑—É–≤–∞–Ω–Ω—è—Ö —î `nil`, –∞–±–æ –±—É–¥—å-—è–∫–∞ –±—É–ª–µ–≤–∞ —É–º–æ–≤–∞ –≤–∏—è–≤–∏—Ç—å—Å—è —Ö–∏–±–Ω–æ—é \(`false`\), –≤—Å—è —É–º–æ–≤–∞ —ñ–Ω—Å—Ç—Ä—É–∫—Ü—ñ—ó `if` –≤–≤–∞–∂–∞—î—Ç—å—Å—è —Ö–∏–±–Ω–æ—é \(`false`\). –ù–∞—Å—Ç—É–ø–Ω—ñ —ñ–Ω—Å—Ç—Ä—É–∫—Ü—ñ—ó `if` —î –µ–∫–≤—ñ–≤–∞–ª–µ–Ω—Ç–Ω–∏–º–∏:

```swift
if let firstNumber = Int("4"), let secondNumber = Int("42"), firstNumber < secondNumber && secondNumber < 100 {
    print("\(firstNumber) < \(secondNumber) < 100")
}
// –ù–∞–¥—Ä—É–∫—É—î "4 < 42 < 100"

if let firstNumber = Int("4") {
    if let secondNumber = Int("42") {
        if firstNumber < secondNumber && secondNumber < 100 {
            print("\(firstNumber) < \(secondNumber) < 100")
        }
    }
}
// –ù–∞–¥—Ä—É–∫—É—î "4 < 42 < 100"
```

> **–ü—Ä–∏–º—ñ—Ç–∫–∞**
>
> –ö–æ–Ω—Å—Ç–∞–Ω—Ç–∏ —ñ –∑–º—ñ–Ω–Ω—ñ, —Å—Ç–≤–æ—Ä–µ–Ω—ñ –∑–∞ –¥–æ–ø–æ–º–æ–≥–æ—é –æ–ø—Ü—ñ–æ–Ω–∞–ª—å–Ω–æ–≥–æ –∑–≤'—è–∑—É–≤–∞–Ω–Ω—è —É —ñ–Ω—Å—Ç—Ä—É–∫—Ü—ñ—ó `if` —î –¥–æ—Å—Ç—É–ø–Ω–∏–º–∏ –ª–∏—à–µ —É —Ç—ñ–ª—ñ —ñ–Ω—Å—Ç—Ä—É–∫—Ü—ñ—ó `if`. –ù–∞ –≤—ñ–¥–º—ñ–Ω—É –≤—ñ–¥ —Ü—å–æ–≥–æ, –∫–æ–Ω—Å—Ç–∞–Ω—Ç–∏ —ñ –∑–º—ñ–Ω–Ω—ñ —Å—Ç–≤–æ—Ä–µ–Ω—ñ —É —ñ–Ω—Å—Ç—Ä—É–∫—Ü—ñ—ó `guard` —î –¥–æ—Å—Ç—É–ø–Ω–∏–º–∏ —É —Ä—è–¥–∫–∞—Ö –∫–æ–¥—É, —â–æ —Å–ª—ñ–¥—É—é—Ç—å –∑–∞ —ñ–Ω—Å—Ç—Ä—É–∫—Ü—ñ—î—é `guard`, —è–∫ –æ–ø–∏—Å–∞–Ω–æ —É —Ä–æ–∑–¥—ñ–ª—ñ [–†–∞–Ω–Ω—ñ–π –≤–∏—Ö—ñ–¥]({% link _book/1_language_guide/4_control_flow.md %}#—Ä–∞–Ω–Ω—ñ–π-–≤–∏—Ö—ñ–¥).

### –û–ø—Ü—ñ–æ–Ω–∞–ª–∏, —â–æ —Ä–æ–∑–≥–æ—Ä—Ç–∞—é—Ç—å—Å—è –Ω–µ—è–≤–Ω–æ

–Ø–∫ –æ–ø–∏—Å–∞–Ω–æ –≤–∏—â–µ, –æ–ø—Ü—ñ–æ–Ω–∞–ª–∏ –≤–∫–∞–∑—É—é—Ç—å –Ω–∞ —Ç–µ, —â–æ –∫–æ–Ω—Å—Ç–∞–Ω—Ç—ñ —á–∏ –∑–º—ñ–Ω–Ω—ñ–π –¥–æ–∑–≤–æ–ª—è—î—Ç—å—Å—è "–Ω–µ –º–∞—Ç–∏ –∑–Ω–∞—á–µ–Ω–Ω—è". –û–ø—Ü—ñ–æ–Ω–∞–ª–∏ –º–æ–∂–Ω–∞ –ø–µ—Ä–µ–≤—ñ—Ä–∏—Ç–∏ –∑–∞ –¥–æ–ø–æ–º–æ–≥–æ—é —ñ–Ω—Å—Ç—Ä—É–∫—Ü—ñ—ó `if`, —â–æ–± –ø–æ–±–∞—á–∏—Ç–∏, —á–∏ —ñ—Å–Ω—É—î –∑–Ω–∞—á–µ–Ω–Ω—è, —ñ –º–æ–∂–Ω–∞ —Ä–æ–∑–≥–æ—Ä–Ω—É—Ç–∏ –∑–∞ –¥–æ–ø–æ–º–æ–≥–æ—é –æ–ø—Ü—ñ–æ–Ω–∞–ª—å–Ω–æ–≥–æ –∑–≤'—è–∑—É–≤–∞–Ω–Ω—è, —â–æ–± –¥–æ—Å—Ç—É–ø–∏—Ç–∏—Å—å –¥–æ –∑–Ω–∞—á–µ–Ω–Ω—è –æ–ø—Ü—ñ–æ–Ω–∞–ª—É, —è–∫—â–æ –≤–æ–Ω–æ —ñ—Å–Ω—É—î.

–Ü–Ω–æ–¥—ñ –∑—ñ —Å—Ç—Ä—É–∫—Ç—É—Ä–∏ –ø—Ä–æ–≥—Ä–∞–º–∏ —è—Å–Ω–æ, —â–æ –æ–ø—Ü—ñ–æ–Ω–∞–ª _–∑–∞–≤–∂–¥–∏_ –º–∞—î –∑–Ω–∞—á–µ–Ω–Ω—è, –ø—ñ—Å–ª—è —Ç–æ–≥–æ —è–∫ –≤–æ–Ω–æ –≤–ø–µ—Ä—à–µ –∑–∞–¥–∞–Ω–æ. –í —Ç–∞–∫–∏—Ö –≤–∏–ø–∞–¥–∫–∞—Ö –∑—Ä—É—á–Ω–æ –ø–æ–∑–±—É—Ç–∏—Å—å –Ω–µ–æ–±—Ö—ñ–¥–Ω–æ—Å—Ç—ñ –ø–µ—Ä–µ–≤—ñ—Ä—è—Ç–∏ —Ç–∞ —Ä–æ–∑–≥–æ—Ä—Ç–∞—Ç–∏ –∑–Ω–∞—á–µ–Ω–Ω—è –æ–ø—Ü—ñ–æ–Ω–∞–ª—É –ø—Ä–∏ –∫–æ–∂–Ω–æ–º—É –¥–æ—Å—Ç—É–ø—ñ –¥–æ –Ω—å–æ–≥–æ, –±–æ –º–æ–∂–Ω–∞ –±–µ–∑–ø–µ—á–Ω–æ –ø—Ä–∏–ø—É—Å—Ç–∏—Ç–∏, —â–æ –æ–ø—Ü—ñ–æ–Ω–∞–ª –∑–∞–≤–∂–¥–∏ –º–∞—î –∑–Ω–∞—á–µ–Ω–Ω—è.
