> –Ø–∫—â–æ –≤–∏ —Å–ø—Ä–æ–±—É—î—Ç–µ –æ—Ç—Ä–∏–º–∞—Ç–∏ —á–∏ –∑–º—ñ–Ω–∏—Ç–∏ –µ–ª–µ–º–µ–Ω—Ç –∑–∞ —ñ–Ω–¥–µ–∫—Å–æ–º, —â–æ –Ω–µ –≤—Ö–æ–¥–∏—Ç—å –≤ —ñ—Å–Ω—É—é—á—ñ –º–µ–∂—ñ –º–∞—Å–∏–≤—É, –≤–∏ —Å–ø—Ä–æ–≤–æ–∫—É—î—Ç–µ –ø–æ–º–∏–ª–∫—É —á–∞—Å—É –≤–∏–∫–æ–Ω–∞–Ω–Ω—è. –í–∏ –º–æ–∂–µ—Ç–µ –ø–µ—Ä–µ–≤—ñ—Ä–∏—Ç–∏, —á–∏ —î –∫–æ—Ä–µ–∫—Ç–Ω–∏–º —ñ–Ω–¥–µ–∫—Å –ø–µ—Ä–µ–¥ –π–æ–≥–æ –≤–∏–∫–æ—Ä–∏—Å—Ç–∞–Ω–Ω—è–º, –ø–æ—Ä—ñ–≤–Ω—è–≤—à–∏ –π–æ–≥–æ —ñ–∑ –≤–ª–∞—Å—Ç–∏–≤—ñ—Å—Ç—é –º–∞—Å–∏–≤—É `count`. –û–∫—Ä—ñ–º –≤–∏–ø–∞–¥–∫—É, –∫–æ–ª–∏ `count` –¥–æ—Ä—ñ–≤–Ω—é—î `0` \(—â–æ –æ–∑–Ω–∞—á–∞—î, —â–æ –º–∞—Å–∏–≤ –ø–æ—Ä–æ–∂–Ω—ñ–π\), –Ω–∞–π–±—ñ–ª—å—à–∏–π –∫–æ—Ä–µ–∫—Ç–Ω–∏–π —ñ–Ω–¥–µ–∫—Å —É –º–∞—Å–∏–≤—ñ –∑–∞–≤–∂–¥–∏ –±—É–¥–µ –¥–æ—Ä—ñ–≤–Ω—é–≤–∞—Ç–∏ `count - 1`, –æ—Å–∫—ñ–ª—å–∫–∏ –º–∞—Å–∏–≤–∏ —ñ–Ω–¥–µ–∫—Å—É—é—Ç—å—Å—è, –ø–æ—á–∏–Ω–∞—é—á–∏ —ñ–∑ –Ω—É–ª—è.

–ü—Ä–∏ –≤–∏–¥–∞–ª–µ–Ω–Ω—ñ –µ–ª–µ–º–µ–Ω—Ç—É –±—É–¥—å-—è–∫—ñ –ø—Ä–æ–≥–∞–ª–∏–Ω–∏ —É –º–∞—Å–∏–≤—ñ –∑–∞–∫—Ä–∏–≤–∞—é—Ç—å—Å—è, —Ç–æ–º—É –∑–Ω–∞—á–µ–Ω–Ω—è –∑–∞ —ñ–Ω–¥–µ–∫—Å–æ–º `0` –∑–Ω–æ–≤—É –¥–æ—Ä—ñ–≤–Ω—é—î `"–®—ñ—Å—Ç—å —è—î—Ü—å"`:

```swift

// firstItem —Ç–µ–ø–µ—Ä –¥–æ—Ä—ñ–≤–Ω—é—î "–®—ñ—Å—Ç—å —è—î—Ü—å"
```

–Ø–∫—â–æ –ø–æ—Ç—Ä—ñ–±–Ω–æ –≤–∏–¥–∞–ª–∏—Ç–∏ –æ—Å—Ç–∞–Ω–Ω—ñ–π –µ–ª–µ–º–µ–Ω—Ç —É –º–∞—Å–∏–≤—ñ, —Å–ª—ñ–¥ –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É–≤–∞—Ç–∏ –º–µ—Ç–æ–¥ `removeLast()` –∑–∞–º—ñ—Å—Ç—å –º–µ—Ç–æ–¥—É `remove(at:)`, —â–æ–± —É–Ω–∏–∫–Ω—É—Ç–∏ –∑–∞–π–≤–æ–≥–æ –∑–≤–µ—Ä—Ç–∞–Ω–Ω—è –¥–æ –≤–ª–∞—Å—Ç–∏–≤–æ—Å—Ç—ñ –º–∞—Å–∏–≤—É `count`. –Ø–∫ —ñ –º–µ—Ç–æ–¥ `remove(at:)`, –º–µ—Ç–æ–¥ `removeLast()` –ø–æ–≤–µ—Ä—Ç–∞—î –≤–∏–¥–∞–ª–µ–Ω–∏–π –µ–ª–µ–º–µ–Ω—Ç:

```swift

// –æ—Å—Ç–∞–Ω–Ω—ñ–π –µ–ª–µ–º–µ–Ω—Ç —É –º–∞—Å–∏–≤—ñ –±—É–ª–æ —â–æ–π–Ω–æ –≤–∏–¥–∞–ª–µ–Ω–æ

// –∫–æ–Ω—Å—Ç–∞–Ω—Ç–∞ apples —Ç–µ–ø–µ—Ä –¥–æ—Ä—ñ–≤–Ω—é—î –≤–∏–¥–∞–ª–µ–Ω–æ–º—É —Ä—è–¥–∫—É "–Ø–±–ª—É–∫–∞"
```

### –Ü—Ç–µ—Ä—É–≤–∞–Ω–Ω—è –º–∞—Å–∏–≤—É

–©–æ–± –ø—Ä–æ—ñ—Ç–µ—Ä—É–≤–∞—Ç–∏ –≤–µ—Å—å –Ω–∞–±—ñ—Ä –µ–ª–µ–º–µ–Ω—Ç—ñ–≤ –º–∞—Å–∏–≤—É, —Å–ª—ñ–¥ –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É–≤–∞—Ç–∏ —Ü–∏–∫–ª `for`-`in`:

```swift
for item in shoppingList {
    print(item)
}
// –®—ñ—Å—Ç—å —è—î—Ü—å
// –ú–æ–ª–æ–∫–æ
// –ë–æ—Ä–æ—à–Ω–æ
// –ü–µ–∫–∞—Ä–Ω–∏–π –ø–æ—Ä–æ—à–æ–∫
// –ë–∞–Ω–∞–Ω–∏
```

–Ø–∫—â–æ –≤–∞–º –ø–æ—Ç—Ä—ñ–±–µ–Ω —Ü—ñ–ª–æ—á–∏—Å–µ–ª—å–Ω–∏–π —ñ–Ω–¥–µ–∫—Å –∫–æ–∂–Ω–æ–≥–æ –µ–ª–µ–º–µ–Ω—Ç–∞ —Ä–∞–∑–æ–º —ñ–∑ –π–æ–≥–æ –∑–Ω–∞—á–µ–Ω–Ω—è–º, —Å–ª—ñ–¥ –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É–≤–∞—Ç–∏ –º–µ—Ç–æ–¥ `enumerated()` —Ç–∞ —ñ—Ç–µ—Ä—É–≤–∞—Ç–∏ –π–æ–≥–æ —Ä–µ–∑—É–ª—å—Ç–∞—Ç. –î–ª—è –∫–æ–∂–Ω–æ–≥–æ –µ–ª–µ–º–µ–Ω—Ç—É –º–∞—Å–∏–≤—É, –º–µ—Ç–æ–¥ `enumerated()` –ø–æ–≤–µ—Ä—Ç–∞—î –∫–æ—Ä—Ç–µ–∂, —â–æ —Å–∫–ª–∞–¥–∞—î—Ç—å—Å—è —ñ–∑ —Ü—ñ–ª–æ–≥–æ —ñ–Ω–¥–µ–∫—Å—É —Ç–∞ –µ–ª–µ–º–µ–Ω—Ç–∞. –Ü–Ω–¥–µ–∫—Å–∏ –ø–æ—á–∏–Ω–∞—é—Ç—å—Å—è —ñ–∑ –Ω—É–ª—è —ñ –∑–±—ñ–ª—å—à—É—é—Ç—å—Å—è –Ω–∞ –æ–¥–∏–Ω–∏—Ü—é —ñ–∑ –∫–æ–∂–Ω–∏–º –µ–ª–µ–º–µ–Ω—Ç–æ–º. –ü—ñ–¥ —á–∞—Å —ñ—Ç–µ—Ä–∞—Ü—ñ—ó, –≤–∏ –º–æ–∂–µ—Ç–µ —Ä–æ–∑–±–∏—Ç–∏ –∫–æ—Ä—Ç–µ–∂ –Ω–∞ —Ç–∏–º—á–∞—Å–æ–≤—ñ –∫–æ–Ω—Å—Ç–∞–Ω—Ç–∏ —á–∏ –∑–º—ñ–Ω–Ω—ñ:

```swift
for (index, value) in shoppingList.enumerated() {
    print("–ï–ª–µ–º–µ–Ω—Ç \(index + 1): \(value)")

// –ï–ª–µ–º–µ–Ω—Ç 1: –®—ñ—Å—Ç—å —è—î—Ü—å
// –ï–ª–µ–º–µ–Ω—Ç 2: –ú–æ–ª–æ–∫–æ
// –ï–ª–µ–º–µ–Ω—Ç 3: –ë–æ—Ä–æ—à–Ω–æ
// –ï–ª–µ–º–µ–Ω—Ç 4: –ü–µ–∫–∞—Ä–Ω–∏–π –ø–æ—Ä–æ—à–æ–∫
// –ï–ª–µ–º–µ–Ω—Ç 5: –ë–∞–Ω–∞–Ω–∏
```

–ë—ñ–ª—å—à–µ —ñ–Ω—Ñ–æ—Ä–º–∞—Ü—ñ—î –ø—Ä–æ —Ü–∏–∫–ª `for`-`in` –º–æ–∂–Ω–∞ –æ—Ç—Ä–∏–º–∞—Ç–∏ —É –ø—ñ–¥—Ä–æ–∑–¥—ñ–ª—ñ [–¶–∏–∫–ª For-In]({% link _book/1_language_guide/4_control_flow.md %}#—Ü–∏–∫–ª-for-in).

## –ú–Ω–æ–∂–∏–Ω–∏

_–ú–Ω–æ–∂–∏–Ω–∏_ –∑–±–µ—Ä—ñ–≥–∞—é—Ç—å –Ω–µ–ø–æ–≤—Ç–æ—Ä–Ω—ñ –∑–Ω–∞—á–µ–Ω–Ω—è –æ–¥–Ω–æ–≥–æ —Ç–∏–ø—É –≤ –Ω–µ–≤–ø–æ—Ä—è–¥–∫–æ–≤–∞–Ω—ñ–π –∫–æ–ª–µ–∫—Ü—ñ—ó. –°–ª—ñ–¥ –∫–æ—Ä–∏—Å—Ç—É–≤–∞—Ç–∏—Å—å –º–Ω–æ–∂–∏–Ω–∞–º–∏ –∑–∞–º—ñ—Å—Ç—å –º–∞—Å–∏–≤—ñ–≤ —É –≤–∏–ø–∞–¥–∫–∞—Ö, –∫–æ–ª–∏ –ø–æ—Ä—è–¥–æ–∫ –µ–ª–µ–º–µ–Ω—Ç—ñ–≤ –Ω–µ —î –≤–∞–∂–ª–∏–≤–∏–º, —á–∏ –∫–æ–ª–∏ —Å–ª—ñ–¥ –±—É—Ç–∏ –ø–µ–≤–Ω–∏—Ö —É —Ç–æ–º—É, —â–æ –µ–ª–µ–º–µ–Ω—Ç–∏ –≤ –∫–æ–ª–µ–∫—Ü—ñ—ó –Ω–µ –ø–æ–≤—Ç–æ—Ä—é—é—Ç—å—Å—è.

> **–ü—Ä–∏–º—ñ—Ç–∫–∞**
>
> –Ü—Å–Ω—É—î –º—ñ—Å—Ç –º—ñ–∂ —Ç–∏–ø–æ–º `Set` —É Swift —Ç–∞ –∫–ª–∞—Å–æ–º `NSSet` —É Foundation. –ó–∞ –¥–µ—Ç–∞–ª—å–Ω—ñ—à–æ—é —ñ–Ω—Ñ–æ—Ä–º–∞—Ü—ñ—î—é –ø—Ä–æ –≤–∏–∫–æ—Ä–∏—Å—Ç–∞–Ω–Ω—è —Ç–∏–ø—É `Set` —ñ–∑ Foundation —Ç–∞ Cocoa, –¥–∏–≤—ñ—Ç—å—Å—è —Ä–æ–∑–¥—ñ–ª "Working with Cocoa Data Types" –≤ –∫–Ω–∏–∑—ñ _Using Swift with Cocoa and Objective-C \(Swift 3.0.1\)_.

### –ó–Ω–∞—á–µ–Ω–Ω—è —Ö–µ—à—É –¥–ª—è –º–Ω–æ–∂–∏–Ω–Ω–∏—Ö —Ç–∏–ø—ñ–≤

–î–ª—è —Ç–∏–ø—É, —â–æ –º–æ–∂–µ –∑–±–µ—Ä—ñ–≥–∞—Ç–∏—Å—å —É –º–Ω–æ–∂–∏–Ω—ñ, –ø–æ–≤–∏–Ω–Ω–∞ –±—É—Ç–∏ –≤–∏–∑–Ω–∞—á–µ–Ω–∞ –æ–ø–µ—Ä–∞—Ü—ñ—è _—Ö–µ—à—É–≤–∞–Ω–Ω—è_. –Ü–Ω—à–∏–º–∏ —Å–ª–æ–≤–∞–º–∏, —Ç–∏–ø –ø–æ–≤–∏–Ω–µ–Ω –¥–∞–≤–∞—Ç–∏ –º–æ–∂–ª–∏–≤—ñ—Å—Ç—å –æ–±—á–∏—Å–ª—é–≤–∞—Ç–∏ –∑–Ω–∞—á–µ–Ω–Ω—è —Å–≤–æ–≥–æ —Ö–µ—à—É. –ó–Ω–∞—á–µ–Ω–Ω—è–º —Ö–µ—à—É —î —Ü—ñ–ª–æ—á–∏—Å–µ–ª—å–Ω–µ –∑–Ω–∞—á–µ–Ω–Ω—è, –∫–æ—Ç—Ä–µ –∑–±—ñ–≥–∞—î—Ç—å—Å—è –¥–ª—è —É—Å—ñ—Ö –æ–¥–Ω–∞–∫–æ–≤–∏—Ö –æ–±'—î–∫—Ç—ñ–≤, —Ç–æ–±—Ç–æ —ñ–∑ `a == b`, —Å–ª—ñ–¥—É—î, —â–æ `a.hashValue == b.hashValue`.

–£—Å—ñ –±–∞–∑–æ–≤—ñ —Ç–∏–ø–∏ —É Swift \(—Ç–∞–∫—ñ —è–∫ `String`, `Int`, `Double`, —Ç–∞ `Bool`\), –º–∞—é—Ç—å –≤–∏–∑–Ω–∞—á–µ–Ω—É –æ–ø–µ—Ä–∞—Ü—ñ—é —Ö–µ—à—É–≤–∞–Ω–Ω—è –∑–∞ –∑–∞–º–æ–≤—á–∞–Ω–Ω—è–º, —ñ –º–æ–∂—É—Ç—å –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É–≤–∞—Ç–∏—Å—å —è–∫ —Ç–∏–ø–∏ –∑–Ω–∞—á–µ–Ω—å —É –º–Ω–æ–∂–∏–Ω–∞—Ö, —á–∏ —è–∫ –∫–ª—é—á—ñ —É —Å–ª–æ–≤–Ω–∏–∫–∞—Ö. –í–∏–ø–∞–¥–∫–∏ —É –ø–µ—Ä–µ—á–∏—Å–ª–µ–Ω–Ω—è—Ö –±–µ–∑ –∞—Å–æ—Ü—ñ–π–æ–≤–∞–Ω–∏—Ö –∑–Ω–∞—á–µ–Ω—å \(—è–∫ –æ–ø–∏—Å–∞–Ω–æ –≤ —Ä–æ–∑–¥—ñ–ª—ñ [–ü–µ—Ä–µ—á–∏—Å–ª–µ–Ω–Ω—è]({% link _book/1_language_guide/7_enumerations.md %})\) —Ç–∞–∫–æ–∂ –≤–∏–∑–Ω–∞—á–∞—é—Ç—å –æ–ø–µ—Ä–∞—Ü—ñ—é —Ö–µ—à—É–≤–∞–Ω–Ω—è –∑–∞ –∑–∞–º–æ–≤—á–∞–Ω–Ω—è–º.

> **–ü—Ä–∏–º—ñ—Ç–∫–∞**
>
> –í–∏ –º–æ–∂–µ—Ç–µ –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É–≤–∞—Ç–∏ –≤–∞—à—ñ –≤–ª–∞—Å–Ω—ñ —Ç–∏–ø–∏ —è–∫ –∑–Ω–∞—á–µ–Ω–Ω—è —É –º–Ω–æ–∂–∏–Ω–∞—Ö —á–∏ —è–∫ –∫–ª—é—á—ñ —É —Å–ª–æ–≤–Ω–∏–∫–∞—Ö, —Ä–µ–∞–ª—ñ–∑—É–≤–∞–≤—à–∏ —É –Ω–∏—Ö –ø—Ä–æ—Ç–æ–∫–æ–ª `Hashable` –∑—ñ —Å—Ç–∞–Ω–¥–∞—Ä—Ç–Ω–æ—ó –±—ñ–±–ª—ñ–æ—Ç–µ–∫–∏ Swift. –¢–∏–ø–∏, —â–æ —Ä–µ–∞–ª—ñ–∑–æ–≤—É—é—Ç—å –ø—Ä–æ—Ç–æ–∫–æ–ª `Hashable`, –ø–æ–≤–∏–Ω–Ω—ñ –Ω–∞–¥–∞—Ç–∏ –≤–ª–∞—Å—Ç–∏–≤—ñ—Å—Ç—å –¥–ª—è —á–∏—Ç–∞–Ω–Ω—è —Ç–∏–ø—É `Int` –Ω–∞ —ñ–º'—è `hashValue`. –ó–Ω–∞—á–µ–Ω–Ω—è, —â–æ –ø–æ–≤–µ—Ä—Ç–∞—î `hashValue` –Ω–µ –æ–±–æ–≤'—è–∑–∫–æ–≤–æ –ø–æ–≤–∏–Ω–Ω–æ –±—É—Ç–∏ –æ–¥–Ω–∞–∫–æ–≤–∏–º –ø—ñ–¥ —á–∞—Å —Ä—ñ–∑–Ω–∏—Ö –∑–∞–ø—É–≤–∫—ñ–≤ –ø—Ä–æ–≥—Ä–∞–º–∏, —á–∏ —É —Ä—ñ–∑–Ω–∏—Ö –ø—Ä–æ–≥—Ä–∞–º–∞—Ö.
>
> –û—Å–∫—ñ–ª—å–∫–∏ –ø—Ä–æ—Ç–æ–∫–æ–ª `Hashable` –Ω–∞—Å–ª—ñ–¥—É—î –ø—Ä–æ—Ç–æ–∫–æ–ª `Equatable`, —Ç–∏–ø–∏, —â–æ —Ä–µ–∞–ª—ñ–∑–æ–≤—É—é—Ç—å –ø—Ä–æ—Ç–æ–∫–æ–ª `Hashable`, –ø–æ–≤–∏–Ω–Ω—ñ —Ç–∞–∫–æ–∂ —Ä–µ–∞–ª—ñ–∑–æ–≤—É–≤–∞—Ç–∏ –ø—Ä–æ–∫–æ—Ç–æ–ª `Equatable`, —Ç–æ–±—Ç–æ –Ω–∞–¥–∞–≤–∞—Ç–∏ —Ä–µ–∞–ª—ñ–∑–∞—Ü—ñ—é –æ–ø–µ—Ä–∞—Ç–æ—Ä—É —Ä—ñ–≤–Ω–æ—Å—Ç—ñ \(`==`\). –ü—Ä–æ—Ç–æ–∫–æ–ª `Equatable` –≤–∏–º–∞–≥–∞—î, —â–æ–± —Ä–µ–∞–ª—ñ–∑–∞—Ü—ñ—è –æ–ø–µ—Ä–∞—Ç–æ—Ä—É `==` –±—É–ª–∞ –≤—ñ–¥–Ω–æ—à–µ–Ω–Ω—è–º —Ä—ñ–≤–Ω–æ—Å—Ç—ñ. –¶–µ –æ–∑–Ω–∞—á–∞—î, —â–æ —Ä–µ–∞–ª—ñ–∑–∞—Ü—ñ—è `==` –ø–æ–≤–∏–Ω–Ω–∞ –∑–∞–¥–æ–≤–æ–ª—å–Ω—è—Ç–∏ —Ç—Ä–∏ –Ω–∞—Å—Ç—É–ø–Ω—ñ —É–º–æ–≤–∏ –¥–ª—è –±—É–¥—å-—è–∫–∏—Ö –∑–Ω–∞—á–µ–Ω—å `a`, `b` —ñ `—Å`:
>
> * `a == a` \(–†–µ—Ñ–ª–µ–∫—Å–∏–≤–Ω—ñ—Å—Ç—å\)
> * –Ü–∑ `a == b` —Å–ª—ñ–¥—É—î `b == a` \(–°–∏–º–µ—Ç—Ä–∏—á–Ω—ñ—Å—Ç—å\)
> * –Ü–∑ `a == b && b == c` —Å–ª—ñ–¥—É—î `a == c` \(–¢—Ä–∞–Ω–∑–∏—Ç–∏–≤–Ω—ñ—Å—Ç—å\)
>
> –ó–∞ –¥–µ—Ç–∞–ª—å–Ω—ñ—à–æ—é —ñ–Ω—Ñ–æ—Ä–º–∞—Ü—ñ—î—é –ø—Ä–æ —Ä–µ–∞–ª—ñ–∑–∞—Ü—ñ—é –ø—Ä–æ—Ç–æ–∫–æ–ª—ñ–≤, –¥–∏–≤—ñ—Ç—å—Å—è —Ä–æ–∑–¥—ñ–ª [–ü—Ä–æ—Ç–æ–∫–æ–ª–∏]({% link _book/1_language_guide/20_protocols.md %}).

### –°–∏–Ω—Ç–∞–∫—Å–∏—Å –º–Ω–æ–∂–∏–Ω

–¢–∏–ø–∏ –º–Ω–æ–∂–∏–Ω —É Swift –∑–∞–ø–∏—Å—É—é—Ç—å—Å—è —è–∫ `Set<Element>`, –¥–µ `Element` —î —Ç–∏–ø–æ–º, –∫–æ—Ç—Ä–∏–π –º–æ–∂–Ω–∞ –∑–±–µ—Ä—ñ–≥–∞—Ç–∏ —É –º–Ω–æ–∂–∏–Ω—ñ. –ù–∞ –≤—ñ–¥–º—ñ–Ω—É –≤—ñ–¥ –º–∞—Å–∏–≤—ñ–≤, –º–Ω–æ–∂–∏–Ω–∏ –Ω–µ –º–∞—é—Ç—å –µ–∫–≤—ñ–≤–∞–ª–µ–Ω—Ç–Ω–æ—ó —Å–∫–æ—Ä–æ—á–µ–Ω–æ—ó —Ñ–æ—Ä–º–∏ –∑–∞–ø–∏—Å—É.

### –°—Ç–≤–æ—Ä–µ–Ω–Ω—è —Ç–∞ —ñ–Ω—ñ—Ü—ñ–∞–ª—ñ–∑–∞—Ü—ñ—è –ø–æ—Ä–æ–∂–Ω—å–æ—ó –º–Ω–æ–∂–∏–Ω–∏

–©–æ–± —Å—Ç–≤–æ—Ä–∏—Ç–∏ –ø–æ—Ä–æ–∂–Ω—é –º–Ω–æ–∂–∏–Ω—É –ø–µ–≤–Ω–æ–≥–æ —Ç–∏–ø—É, —Å–ª—ñ–¥ –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É–≤–∞—Ç–∏ –Ω–∞—Å—Ç—É–ø–Ω–∏–π —Å–∏–Ω—Ç–∞–∫—Å–∏—Å:

```swift

print("letters –º–∞—î —Ç–∏–ø Set<Character> —ñ –º—ñ—Å—Ç–∏—Ç—å \(letters.count) –µ–ª–µ–º–µ–Ω—Ç—ñ–≤.")
// –ù–∞–¥—Ä—É–∫—É—î "letters –º–∞—î —Ç–∏–ø  Set<Character> —ñ –º—ñ—Å—Ç–∏—Ç—å 0 –µ–ª–µ–º–µ–Ω—Ç—ñ–≤."
```

> **–ü—Ä–∏–º—ñ—Ç–∫–∞**
>

```swift
// –ù–∞–¥—Ä—É–∫—É—î "–ù–∞–∑–≤–∞ –≤–∏–¥–∞–ª–µ–Ω–æ–≥–æ –∞–µ—Ä–æ–ø–æ—Ä—Ç—É: Dublin Airport."
```

### –Ü—Ç–µ—Ä—É–≤–∞–Ω–Ω—è —Å–ª–æ–≤–Ω–∏–∫–∞

–©–æ–± –ø—Ä–æ—ñ—Ç–µ—Ä—É–≤–∞—Ç–∏ –≤—Å—ñ –ø–∞—Ä–∏ –∫–ª—é—á-–∑–Ω–∞—á–µ–Ω–Ω—è —Å–ª–æ–≤–Ω–∏–∫–∞, —Å–ª—ñ–¥ –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É–≤–∞—Ç–∏ —Ü–∏–∫–ª `for`-`in`. –ö–æ–∂–µ–Ω –µ–ª–µ–º–µ–Ω—Ç —É —Å–ª–æ–≤–Ω–∏–∫—É –ø—Ä–∏ —Ü—å–æ–º—É –±—É–¥–µ –ø–æ–≤–µ—Ä—Ç–∞—Ç–∏—Å—å —è–∫ –∫–æ—Ä—Ç–µ–∂ `(key, value)`, —ñ –º–æ–∂–Ω–∞ –æ–ø–∏—Å–∞—Ç–∏ –¥–µ–∫–æ–º–ø–æ–∑–∏—Ü—ñ—é –π–æ–≥–æ –µ–ª–µ–º–µ–Ω—Ç—ñ–≤ —É —Ç–∏–º—á–∞—Å–æ–≤—ñ –∫–æ–Ω—Å—Ç–∞–Ω—Ç–∏ —á–∏ –∑–º—ñ–Ω–Ω—ñ —è–∫ —á–∞—Å—Ç–∏–Ω—É —ñ—Ç–µ—Ä–∞—Ü—ñ—ó:

```swift
for (airportCode, airportName) in airports {
    print("\(airportCode): \(airportName)")
}
// YYZ: Toronto Pearson
// LHR: London Heathrow
```

–ë—ñ–ª—å—à–µ —ñ–Ω—Ñ–æ—Ä–º–∞—Ü—ñ—î –ø—Ä–æ —Ü–∏–∫–ª `for`-`in` –º–æ–∂–Ω–∞ –æ—Ç—Ä–∏–º–∞—Ç–∏ —É –ø—ñ–¥—Ä–æ–∑–¥—ñ–ª—ñ [–¶–∏–∫–ª For-In]({% link _book/1_language_guide/4_control_flow.md %}#—Ü–∏–∫–ª-for-in).

–¢–∞–∫–æ–∂ –º–æ–∂–Ω–∞ –æ—Ç—Ä–∏–º–∞—Ç–∏ –∫–æ–ª–µ–∫—Ü—ñ—ó –æ–∫—Ä–µ–º–æ –∫–ª—é—á—ñ–≤ —Ç–∞ –∑–Ω–∞—á–µ–Ω—å —Å–ª–æ–≤–Ω–∏–∫–∞ –∑–∞ –¥–æ–ø–æ–º–æ–≥–æ—é –π–æ–≥–æ –≤–ª–∞—Å—Ç–∏–≤–æ—Å—Ç–µ–π `keys` —Ç–∞ `values`, –¥–ª—è –ø–æ–¥–∞–ª—å—à–æ—ó —ó—Ö —ñ—Ç–µ—Ä–∞—Ü—ñ—ó:

```swift

    print("–ö–æ–¥ –∞–µ—Ä–æ–ø–æ—Ä—Ç—É: \(airportCode)")

// –ö–æ–¥ –∞–µ—Ä–æ–ø–æ—Ä—Ç—É: LHR

    print("–Ü–º'—è –∞–µ—Ä–æ–ø–æ—Ä—Ç—É: \(airportName)")

// –Ü–º'—è –∞–µ—Ä–æ–ø–æ—Ä—Ç—É: Toronto Pearson
// –Ü–º'—è –∞–µ—Ä–æ–ø–æ—Ä—Ç—É: London Heathrow
```

–Ø–∫—â–æ –ø–æ—Ç—Ä—ñ–±–Ω–æ –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É–≤–∞—Ç–∏ –∫–ª—é—á—ñ —á–∏ –∑–Ω–∞—á–µ–Ω–Ω—è —Å–ª–æ–≤–Ω–∏–∫–∞ –∑ API, —â–æ –ø—Ä–∏–π–º–∞—î –Ω–∞ –≤—Ö—ñ–¥ –º–∞—Å–∏–≤ \(`Array`\), —Å–ª—ñ–¥ –ø—Ä–æ—ñ–Ω—ñ—Ü—ñ–∞–ª—ñ–∑—É–≤–∞—Ç–∏ –Ω–æ–≤–∏–π –º–∞—Å–∏–≤ –∑–Ω–∞—á–µ–Ω–Ω—è–º –≤–ª–∞—Å—Ç–∏–≤–æ—Å—Ç—ñ `keys` —á–∏ `values` –≤—ñ–¥–ø–æ–≤—ñ–¥–Ω–æ:

```swift
// airportCodes –¥–æ—Ä—ñ–≤–Ω—é—î ["YYZ", "LHR"]
```
